CSS Minification Fix — Issue #18795

The regex-based minifyCss in scripts/build-minified-css.mjs breaks complex CSS values. Below are examples that would be mangled by the old minifier but are safe with an AST-based tool like lightningcss.

grid-template-areas (quoted strings)

Broken Regex merges "a b" "c d" into "a b""c d" 
grid-template-areas:
  "header header header"
  "sidebar main   main"
  "footer footer footer";

Broken Regex strips spaces: calc(100% - 2rem) → calc(100%-2rem)
