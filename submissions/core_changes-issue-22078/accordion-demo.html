What is EaseMotion CSS?
EaseMotion CSS is an animation-first CSS framework with reusable UI components, modern effects, and zero dependencies. It provides utility classes, component styles, and entrance animations out of the box.
How do I install it?
You can use it via CDN: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css@latest/easemotion.min.css" /> or install via npm with npm install easemotion-css.
Is it compatible with other frameworks?
Yes! EaseMotion CSS is framework-agnostic. It works with React, Vue, Angular, Svelte, or plain HTML. The class-based approach means you can drop it into any project without configuration.
Do I need to write custom keyframes?
Not at all. EaseMotion includes 50+ pre-built animation classes. Just add ease-slide-up, ease-float, ease-pulse, and more to any element.
