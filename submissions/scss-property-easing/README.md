# SCSS @property Custom Easing Mixins

This submission adds reusable SCSS mixins for defining CSS `@property`-based custom easing curves and animation helpers.

## Features

- Register custom CSS properties
- Create reusable cubic-bezier easing variables
- Transition mixin
- Hover scale effect
- Hover rotate effect
- Hover lift effect
- Fade-in animation helper
