SCSS Ripple Effect Mixins

A lightweight, modular SCSS ripple mixin driving performance-first click ripples purely using CSS keyframe animations.
Standard Center Ripple
The default ripple compiles with light reflections expanding outwards from the exact center of the component.
Directional Corner Ripples
Configurable coordinates origins trigger ripples expanding outward from specific button corners.
Custom Easing & Color Variants
Supports parameter adjustments for color values (dark ripples for white components), speeds, and sizes.
