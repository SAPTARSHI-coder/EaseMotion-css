# Elevation Shadow Utility (SCSS)

A reusable SCSS utility module that provides configurable multi-layer box-shadow mixins for creating consistent elevation effects in the EaseMotion CSS design system.

---

## 📌 Description

This SCSS module helps developers apply consistent elevation (depth) effects using predefined shadow levels. It improves UI hierarchy, reduces repetitive shadow code, and ensures a clean and scalable design system approach.

---

## ✨ Features

- Multi-layer box-shadow elevation system
- 5 predefined elevation levels (1–5)
- Hover elevation support
- Smooth transition utility
- Fully reusable SCSS mixins
- Lightweight and customizable

---

## 📁 Installation

Place the `_elevation-shadow.scss` file into your SCSS utilities folder and import it:

```scss
@import "elevation-shadow";
