# Precision Auto Care — Auto Repair Shop Landing Page

1. **What does this do?**  
   A trustworthy, professional full-page business landing page for an auto repair shop, built entirely with EaseMotion CSS utility classes and component classes — no custom keyframes.

2. **How is it used?**  
   Open `demo.html` directly in any browser. The page links to EaseMotion CSS via CDN and uses only existing `ease-*` classes for all animations, layout, and interactions.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can build complete, production-quality automotive business landing pages with zero custom animations and no build step. This auto-shop theme uses a dark industrial aesthetic with red accents for a trustworthy, technical feel.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Full-viewport dark gradient hero with radial glow, entrance animations (`ease-slide-up`, `ease-delay-*`), red accent text, pulsing CTA |
| Services | 6 service cards with icons (oil, brakes, engine, AC, tires, diagnostics) and `ease-card-lift` hover |
| Why Choose Us | 4 stat cards (35+ years, 15k+ cars, 4.9 rating, 12 techs) + 3 value proposition cards |
| Pricing & Specials | 3 pricing cards (Oil Change $29.95, Brakes $149, Tune-Up $199) with featured "Most Popular" plan |
| Customer Reviews | 3 review cards with vehicle badges and star ratings |
| Appointment Form | 2-column layout: booking form (name, phone, vehicle, service, date) + Location/Hours info cards |
| Footer | 4-column footer with brand, services, quick links, payment info |
