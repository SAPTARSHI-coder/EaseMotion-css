# Accordion component

CSS-only accordion with smooth expand/collapse. Exclusive mode via `<input type="radio">` — only one open at a time.

- indigo/violet palette, system font
- all demo styles in `<style>` tag
- `style.css` only contains accordion classes
- keyboard accessible, reduced-motion support

## Files

- `demo.html` — FAQ accordion with 4 items, accessibility notes, usage code
- `style.css` — accordion styles, transitions, focus, reduced-motion
- `README.md` — this file
