# Framework Integration Guides

## Overview

Integration guides for using EaseMotion CSS with React, Vue, and Astro. Covers installation, global import, component examples, SSR handling, and scoped styles.

## Files

| File | Framework | Topics |
|------|-----------|--------|
| `framework-integration-react.md` | React + Next.js | App Router, Pages Router, SSR, CSS Modules |
| `framework-integration-vue.md` | Vue + Nuxt 3 | Global import, scoped styles, Nuxt config |
| `framework-integration-astro.md` | Astro | Layout imports, SSR, scoped styles |

## Usage

Each guide is self-contained. Open the `.md` file directly or serve via GitHub Pages.

## Why This Matters

Framework integration guides reduce onboarding friction. Developers moving from utility-first frameworks need clear patterns for component-based usage, SSR compatibility, and style isolation.

## Goal

Help developers use EaseMotion CSS confidently in any modern JavaScript framework.
