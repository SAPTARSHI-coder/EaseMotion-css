What is EaseMotion CSS?

EaseMotion CSS is an animation-first CSS framework with reusable UI components, modern effects, and zero dependencies. It is designed to be lightweight, beginner-friendly, and fully open-source.

Built with modern CSS features like custom properties, layers, and container queries, EaseMotion provides a consistent design system for rapid prototyping and production use alike. Every component and utility class is crafted to work seamlessly together.

Core Philosophy

The framework follows three core principles: performance-first animations using GPU-accelerated properties (transform and opacity), zero JavaScript dependencies for base functionality, and full accessibility support including reduced-motion preferences.

Every animation utility respects the user's system preferences, and all interactive components are keyboard-navigable with proper ARIA attributes out of the box.

Getting Started

Add the following link to your HTML document to get started:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SAPTARSHI-coder/EaseMotion-css@main/easemotion.css">

Then simply add any EaseMotion class to your elements. For example, add ease-fade-in to a heading, ease-slide-up to a card, or wrap content in an ease-reveal container for scroll-triggered entrance.

Customize everything using CSS custom properties. Change --ease-speed-medium to adjust animation durations, --ease-color-primary to match your brand colors, and --ease-space-4 to control spacing.

Components Library

EaseMotion includes a growing library of UI components: accordions, modals, tooltips, dropdowns, drawers, navigation bars, cards, buttons, badges, progress bars, steppers, carousels, and more.

Each component follows BEM-like naming conventions, uses CSS layers for style isolation, and includes proper reduced-motion fallbacks. Components are designed to be modular — use only what you need.

Animation Utilities

Over 60 animation utility classes are available, including entrance animations (fade, slide, zoom, bounce, roll, light-speed), emphasis animations (pulse, swing, wobble, jello, flash, rubber-band), and exit animations (fade-out, slide-out, hinge, roll-out, light-speed-out).

Speed modifiers (ease-fast, ease-slow) and iteration modifiers (ease-infinite) work across all animation classes, giving you fine-grained control over timing.

Browser Support

EaseMotion CSS supports all modern browsers including Chrome 90+, Firefox 88+, Safari 14+, and Edge 90+. CSS custom property fallbacks ensure graceful degradation in older browsers.

Animation classes that use newer CSS features like @container or grid-template-rows animation include fallback behavior for browsers that do not yet support these features.
