How it works

The water level rises with a smooth wave animation. The glass uses a clip-path to create the tapered shape, and the water surface has a CSS wave effect using a pseudo-element with a sine-like border-radius animation.

@keyframes wave {
  0%   { border-radius: 45% 55% 60% 40% / 50% 45% 55% 50%; }
  50%  { border-radius: 55% 45% 40% 60% / 45% 55% 50% 55%; }
  100% { border-radius: 45% 55% 60% 40% / 50% 45% 55% 50%; }
}
