Card Glow · Reduced Motion Bug

.ease-card-glow under prefers-reduced-motion: reduce sets transition: none but the hover glow still snaps instantly — a sudden jarring flash.

Bug transition: none

⚡

Broken

With reduced motion, glow snaps on instantly — worse than a gradual transition.

The glow appears as a 0ms flash on hover.

Fixed box-shadow: none

✨

Fixed

Glow is removed entirely on hover under reduced motion — no sudden change.

No flash. The glow effect is suppressed on interaction.

Fix

Instead of transition: none (which still lets the hover box-shadow snap), suppress the glow itself:

/* preferred approach */
@media (prefers-reduced-motion: reduce) {
  .ease-card-glow:hover,
  .ease-card-glow:focus-visible {
    box-shadow: none;
  }
}

/* alternative: keep glow always-on so no transition occurs */
@media (prefers-reduced-motion: reduce) {
  .ease-card-glow {
    box-shadow: 0 0 20px rgba(251, 191, 36, 0.3);
  }
}
