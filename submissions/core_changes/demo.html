A zero-dependency, animation-first CSS framework with readable class names. No build step needed.

Add the CDN link to your <head> or run npm install easemotion-css. Works immediately.

Yes. Works with any framework. Import the stylesheet and use classes directly in className.

Override CSS custom properties like --ease-color-primary to theme the entire framework.

Accessibility

  • tab to focus the radio input
  • space/enter to toggle
  • label linked to input via for attribute
  • focus-visible outline on interactive elements
  • reduced-motion disables transitions

Usage

<div class="acc">
  <div class="acc-it">
    <input type="radio" name="x" id="q1" class="acc-trg" />
    <label class="acc-lbl" for="q1">Question?</label>
    <div class="acc-cnt">
      <p>Answer.</p>
    </div>
  </div>
</div>
