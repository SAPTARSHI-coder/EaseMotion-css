Only one item open at a time using <input type="radio"> with shared name.

Usage

<div class="accordion">
  <div class="accordion-item">
    <input type="radio" name="acc" id="q1" class="accordion-trigger" />
    <label class="accordion-header" for="q1">Question?</label>
    <div class="accordion-content">
      <p>Answer here.</p>
    </div>
  </div>
</div>

Uses max-height transition for smooth expand/collapse. The radio input pattern ensures exclusive open — only one item can be open at a time.

Accessibility

  • <label> linked to <input> via for attribute — click or tap anywhere on the header to toggle
  • Keyboard: Tab to focus the radio input, Space/Enter to toggle
  • CSS-only — no JavaScript required for the toggle behavior
  • Focus-visible outlines on interactive elements
