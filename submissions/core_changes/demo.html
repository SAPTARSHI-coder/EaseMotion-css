Expandable Search

Animated search input that expands from 44px to 280px on focus with icon fade-in

Demo

Click or tab into the search icon below to see it expand.

🔍

How it works

The wrapper uses inline-flex with width transitioning from 44px to 280px on :focus-within. The icon is absolutely positioned inside the wrapper, and the input fades in as the wrapper expands.

Usage

<div class="search-wrapper">
  <span class="search-icon">&#128269;</span>
  <input type="text" class="search-input"
         placeholder="Search..." />
</div>

The :focus-within pseudo-class on the wrapper triggers the width expansion and input opacity transition.

Behavior

  • Collapsed: shows only the search icon (44px circle)
  • Focus: expands to 280px, input fades in, icon shifts left
  • Blur: collapses back to icon-only if empty
  • Keyboard accessible via Tab
  • Respects prefers-reduced-motion
