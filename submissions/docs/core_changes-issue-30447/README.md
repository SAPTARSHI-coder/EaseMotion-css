# Floating Label Input — Issue #30447

Material-design inspired floating label form inputs.

## Classes

| Class | Purpose |
|-------|---------|
| `.ease-floating-input` | Input wrapper |
| `.ease-floating-label` | Floating label |
| `.ease-floating-input-filled` | Persistent float state |
| `.ease-floating-input-error` | Error state (red border) |
| `.ease-floating-input-success` | Success state (green border) |

## CSS Variables

- `--ease-fi-padding`, `--ease-fi-border`, `--ease-fi-border-focus`
- `--ease-fi-radius`, `--ease-fi-label-color`, `--ease-fi-label-active`
- `--ease-fi-duration`

Uses `:placeholder-shown` selector for CSS-only float detection.
