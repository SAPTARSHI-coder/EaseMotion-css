Features

IntersectionObserver tracks which section is most visible and applies the .active class to the corresponding nav link. The sidebar uses position: sticky with top: 2rem so it follows the viewport.

Each nav link calls element.scrollIntoView({ behavior: 'smooth' }) for fluid scrolling.
