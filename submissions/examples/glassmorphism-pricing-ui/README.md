# GlassMorph Pricing UI Kit

A modern GlassMorph Pricing UI Kit built using **pure HTML and CSS**. The project showcases animated glassmorphism pricing cards with responsive layouts and elegant visual effects.

---

## ✨ Features

- Pure HTML & CSS
- Responsive Design
- Glassmorphism Cards
- Animated Gradient Background
- Floating Background Blobs
- Four Pricing Plans
- Featured "Pro" Plan
- Monthly / Yearly Toggle (UI)
- Feature Comparison Table
- Testimonials Section
- FAQ Section
- Call-to-Action Section
- Hover Effects
- Shine Animation
- Glow Effects
- Floating Card Animation
- Keyboard Accessible
- Reduced Motion Support
