# Scroll To Top Button

## What does this do?

This submission demonstrates a simple scroll-to-top button that appears after the user scrolls down the page and smoothly returns the page to the top when clicked.

## How is it used?

Open `demo.html` in a browser, scroll down until the button appears, and click the button to smoothly return to the top of the page.

## Why is it useful?

Scroll-to-top controls improve usability on long pages by giving users a fast way to return to the beginning without manually scrolling back up.
