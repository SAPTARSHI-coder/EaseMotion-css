view-timeline-inset

Offset adjustment for scroll-driven animation triggers — compare 0px, 100px, and -50px

inset: 0px

Animation activates exactly when the element enters the viewport — no offset.

Card A1

Default behavior. Fires at the viewport edge.

Card A2

No offset applied to the trigger window.

inset: 100px

Animation delays — element must be 100px inside the viewport before triggering.

Card B1

Delayed reveal. Waits until 100px inside.

Card B2

Positive inset pushes the start boundary inward.

inset: -50px

Animation fires before the element enters the viewport — a pre-emptive trigger.

Card C1

Pre-emptive reveal. Starts 50px before viewport.

Card C2

Negative inset extends the boundary outward.
