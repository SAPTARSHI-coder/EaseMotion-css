# Text Rendering Utilities

CSS utility classes for the `text-rendering` property.

## Usage

```html
<div class="text-rendering-auto">...</div>
```

## Classes

- `.text-rendering-auto` — text-rendering: auto;
- `.text-rendering-speed` — text-rendering: optimizeSpeed;
- `.text-rendering-legibility` — text-rendering: optimizeLegibility;
- `.text-rendering-precision` — text-rendering: geometricPrecision;
- `.text-optimize-speed` — text-rendering: optimizeSpeed;
- `.text-optimize-legibility` — text-rendering: optimizeLegibility;
- `.text-optimize-precision` — text-rendering: geometricPrecision;
- `.render-auto` — text-rendering: auto;
- `.render-speed` — text-rendering: optimizeSpeed;
- `.render-legibility` — text-rendering: optimizeLegibility;
- `.render-precision` — text-rendering: geometricPrecision;
- `.text-render-auto` — text-rendering: auto;
- `.text-render-speed` — text-rendering: optimizeSpeed;
- `.text-render-legibility` — text-rendering: optimizeLegibility;
- `.text-render-precision` — text-rendering: geometricPrecision;
- `.text-render-inherit` — text-rendering: inherit;
- `.text-render-initial` — text-rendering: initial;
- `.text-render-unset` — text-rendering: unset;
- `.text-rendering-inherit` — text-rendering: inherit;
- `.text-rendering-initial` — text-rendering: initial;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
