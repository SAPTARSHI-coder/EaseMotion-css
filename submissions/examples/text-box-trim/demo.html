/* Remove leading/trailing text spacing */
.trimmed-element {
  text-box-trim: trim-both;
  text-box-edge: cap alphabetic;
}

/* Aligns the cap height and alphabetic baseline
   to the content box edges, eliminating the
   font's internal leading space. */

How it works

Every font includes built-in leading (line gap) above the cap height and below the alphabetic baseline. This space is essential for readable line wrapping but problematic for precise UI alignment — buttons, cards, and headings often look misaligned vertically. text-box-trim: trim-both removes this overshoot so the cap height starts at the content box top and the alphabetic baseline ends at the content box bottom, giving pixel-perfect vertical rhythm without manual offset hacks.
