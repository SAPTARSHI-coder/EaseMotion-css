# Animated Dropdown Menu

A CSS-only multi-level dropdown navigation menu using the checkbox hack. Features hover and click-to-expand menus, submenu support, and smooth opacity + translate transitions on open/close.

**Created by @saurabhhhcodes**

## Features

- Pure CSS — no JavaScript required
- Checkbox hack for dropdown toggling
- Hover support for desktop dropdown expansion
- Multi-level submenus (2 levels deep)
- Smooth opacity + translateY fade-in animation
- Glassmorphism sticky navbar with backdrop blur
- CTA button with gradient styling
- Responsive mobile layout
- Hover effects on menu items with slide padding
