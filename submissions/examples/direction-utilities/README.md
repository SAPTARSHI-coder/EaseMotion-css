# Direction Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/direction-utilities/style.css">
<div class="direction-utilities-card direction-utilities-card-featured">Direction Utilities Content</div>
```

## Classes
- `.direction-utilities-card`: Primary card with breathe animation
- `.direction-utilities-card-alt`: Secondary variant
- `.direction-utilities-card-ghost`: Ghost/outline card
- `.direction-utilities-card-glow`: Glow effect card
- `.direction-utilities-card-featured`: Gradient featured card
- `.direction-utilities-card-sm / -md / -lg`: Size variants
- `.direction-utilities-v-1 through -5`: Opacity variants
- `.direction-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.direction-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 206deg
- Border radius: 7px / 15px / 23px
