# Developer Portfolio Hero Section

A clean, aesthetic landing hero section built for UI architects and frontend developers.

## Design Highlights
- **High-Fidelity Typography:** Clear font pairings using `Outfit` and `JetBrains Mono`.
- **Gradients:** Soft gradient masking overlays.
- **Interactions:** Subtle vertical shifts on button and skill-tag hover triggers.
