# CSS Bottom Navigation

## What does this do?
Mobile bottom navigation bars with icon-label, badge indicators, FAB button, and minimal variants.

## How is it used?
```html
<nav class="bn-bar"><a href="#" class="bn-item bn-active"><span class="bn-icon">&#127968;</span><span class="bn-label">Home</span></a></nav>
```

## Why is it useful?
Mobile-first bottom nav. Active state, badge notifications, center FAB button, icon-only minimal mode.
