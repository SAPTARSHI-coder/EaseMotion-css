# Personalized Gifts E-commerce

A thoughtful e-commerce landing page for custom gift creation.

## EaseMotion Classes Showcased
- `ease-fade-in`: Entrance animation for hero and reviews.
- `ease-slide-up`: Scroll-reveal for process timeline and products.
- `ease-hover-scale`: Interaction for call-to-action buttons.

## How to Run
Open `demo.html` directly in your browser. No build steps required.
