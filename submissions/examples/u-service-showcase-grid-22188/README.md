# ease-service-showcase-grid

Animated services showcase with layered hover effects built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive service grid
- Layered hover animation
- Animated service icons
- Call-to-action buttons
- Accessibility support
- prefers-reduced-motion support

## Files

- demo.html
- style.css

## Usage

```html
<section class="services">
...
</section>
```

Hover over a service card to reveal the animated overlay, icon motion, and action button.
