# Pointer Events Utilities

## Overview
Grid cell utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/pointer-events-utilities/style.css">
<div class="pointer-events-utilities-grid-wrap pointer-events-utilities-cols-3 pointer-events-utilities-gap-2">
  <div class="pointer-events-utilities-cell">Pointer Events Utilities Cell</div>
</div>
```

## Classes
- `.pointer-events-utilities-cell`: Primary grid cell
- `.pointer-events-utilities-cell-alt`: Secondary variant
- `.pointer-events-utilities-cell-accent`: Accent variant
- `.pointer-events-utilities-cell-featured`: Gradient featured cell
- `.pointer-events-utilities-cell-outline`: Outline variant
- `.pointer-events-utilities-cell-sm / -md / -lg / -xl`: Size variants
- `.pointer-events-utilities-cols-2 / -3 / -4`: Column count
- `.pointer-events-utilities-gap-1 / -2 / -3`: Gap variants
- `.pointer-events-utilities-rwd-1 through -3`: Responsive padding
