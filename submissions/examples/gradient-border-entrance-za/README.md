# Gradient Border Entrance Animation

A CSS-only gradient border card that reveals itself with a sweeping entrance animation. The border uses a conic gradient with continuous hue rotation.

## Files

- `demo.html` - Card with gradient border
- `style.css` - Border reveal animation, hue rotation, card layout

## Usage

Open `demo.html` in a browser to see the gradient border animate into view and continuously shift colors.
