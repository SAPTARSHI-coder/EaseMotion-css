# Clip Path Animate

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/clip-path-animate/style.css">
<li class="clip-path-animate-row clip-path-animate-row-active">
  Clip Path Animate item <span class="clip-path-animate-tag">Clip Path Animate</span>
</li>
```

## Classes
- `.clip-path-animate-row`: List row with hover
- `.clip-path-animate-row-active`: Active row with slide-in animation
- `.clip-path-animate-tag / -alt / -outline / -success / -warn`: Tag variants
- `.clip-path-animate-chip / -primary / -success`: Chip variants
- `.clip-path-animate-hl`: Highlight text
- `.clip-path-animate-prio-high / -med / -low`: Priority indicators
- `.clip-path-animate-idx-1 through -3`: Index numbering styles
