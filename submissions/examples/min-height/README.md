# Min Height Utilities

CSS utility classes for the `min-height` property.

## Usage

```html
<div class="min-h-0">...</div>
```

## Classes

- `.min-h-0` — min-height: 0;
- `.min-h-full` — min-height: 100%;
- `.min-h-screen` — min-height: 100vh;
- `.min-h-min` — min-height: min-content;
- `.min-h-max` — min-height: max-content;
- `.min-h-fit` — min-height: fit-content;
- `.min-h-auto` — min-height: auto;
- `.min-h-1` — min-height: 0.25rem;
- `.min-h-2` — min-height: 0.5rem;
- `.min-h-4` — min-height: 1rem;
- `.min-h-8` — min-height: 2rem;
- `.min-h-16` — min-height: 4rem;
- `.min-h-32` — min-height: 8rem;
- `.min-h-48` — min-height: 12rem;
- `.min-h-64` — min-height: 16rem;
- `.min-h-screen-sm` — min-height: 640px;
- `.min-h-screen-md` — min-height: 768px;
- `.min-h-screen-lg` — min-height: 1024px;
- `.min-h-screen-xl` — min-height: 1280px;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
