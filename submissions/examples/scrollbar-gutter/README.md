# Scrollbar Gutter

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/scrollbar-gutter/style.css">
<li class="scrollbar-gutter-row scrollbar-gutter-row-active">
  Scrollbar Gutter item <span class="scrollbar-gutter-tag">Scrollbar Gutter</span>
</li>
```

## Classes
- `.scrollbar-gutter-row`: List row with hover
- `.scrollbar-gutter-row-active`: Active row with slide-in animation
- `.scrollbar-gutter-tag / -alt / -outline / -success / -warn`: Tag variants
- `.scrollbar-gutter-chip / -primary / -success`: Chip variants
- `.scrollbar-gutter-hl`: Highlight text
- `.scrollbar-gutter-prio-high / -med / -low`: Priority indicators
- `.scrollbar-gutter-idx-1 through -3`: Index numbering styles
