# Animated Contact Form

A CSS-only animated contact form with floating labels, real-time validation states (success/error), and a submit button with hover animation. Fields include name, email, subject, and message textarea.

**Created by @saurabhhhcodes**

## Features

- Pure CSS — no JavaScript required
- Floating label animation using `:placeholder-shown` and `:focus` selectors
- Real-time validation indicators (✓ for valid, ! for invalid)
- Success/error state styling with color-coded indicators
- Animated submit button with lift, scale, and icon slide effects
- Glassmorphism container design
- Responsive two-column name/email layout
- Privacy policy checkbox
- Smooth transitions on all interactive elements
