# Shape Margin

## Overview
List item and badge utilities for shape-margin in EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/shape-margin/style.css">
<li class="shape-margin-item shape-margin-item-on">
  Item content
  <span class="shape-margin-badge shape-margin-badge-pri">Label</span>
</li>
```

## Classes
- `.shape-margin-item` through `.shape-margin-item-13`: List items
- `.shape-margin-badge-pri / -suc / -war / -out`: Badge variants
- `.shape-margin-hl`: Text highlight effect
- `.shape-margin-tag`: Uppercase tag label
- `.shape-margin-pri-1 through -6`: Left border priority indicators
