# Shape Margin

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/shape-margin/style.css">
<li class="shape-margin-row shape-margin-row-active">
  Shape Margin item <span class="shape-margin-tag">Shape Margin</span>
</li>
```

## Classes
- `.shape-margin-row`: List row with hover
- `.shape-margin-row-active`: Active row with slide-in animation
- `.shape-margin-tag / -alt / -outline / -success / -warn`: Tag variants
- `.shape-margin-chip / -primary / -success`: Chip variants
- `.shape-margin-hl`: Highlight text
- `.shape-margin-prio-high / -med / -low`: Priority indicators
- `.shape-margin-idx-1 through -3`: Index numbering styles
