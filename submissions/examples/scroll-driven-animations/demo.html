Reference

Class Reference

All available utility classes and their purpose.

Class Effect Timeline type
.em-scroll-view Binds animation to element entry into viewport view()
.em-scroll-timeline Binds animation to root page scroll position scroll(root)
.em-progress-bar Fixed reading-progress indicator bar styles — component
.em-fade-up / down / left / right Fade + translate entrance use with .em-scroll-view
.em-scale-up / down Fade + scale entrance use with .em-scroll-view
.em-rotate-in Fade + slight rotation entrance use with .em-scroll-view
.em-flip-x / flip-y 3D perspective flip entrance use with .em-scroll-view
.em-blur-in Fade + blur-to-focus entrance use with .em-scroll-view
.em-reveal-clip Horizontal clip-path sweep reveal use with .em-scroll-view
.em-parallax-slow Slow vertical parallax shift use with .em-scroll-timeline
