ease-scroll-*

CSS Scroll-Driven Animation utilities — animate elements based on scroll position and viewport visibility. Zero JavaScript required for modern browsers.

⚡ Requires Chrome 115+ or Edge 115+. Other browsers use the IntersectionObserver fallback included in this demo.
Scroll down to see the effects in action.

ease-scroll-reveal

Scroll Reveal

Elements fade in and slide up as they enter the viewport. The ease-scroll-reveal class animates during the entry phase of the scroll timeline.

🌱

Fade In Up

Appears smoothly as you scroll down with a subtle upward motion.

.ease-scroll-reveal
✨

Staggered

Each card animates with a slight delay, creating a cascade effect.

animation-delay: 0.15s
🚀

Delayed

Longer delay for a wave-like reveal across the grid.

animation-delay: 0.3s

ease-view-reveal + ease-view-scale-in

View-Triggered Animations

These classes use view() timeline and cover range for a fuller animation that continues as the element stays in view.

👋

View Reveal

Fade-in-up animation that plays across the entry to cover half of the viewport.

.ease-view-reveal
🔍

Scale In

Scales from 0.8 to 1.0 as the element enters the viewport.

.ease-view-scale-in

Blur & Slide Effects

Creative scroll-driven effects: unblur as elements scroll in, slide in from left or right.

Slide In

Quick Range

ease-scroll-parallax + ease-scroll-zoom

Parallax & Zoom

Background-style effects linked to the root scroll position.

Parallax

This section moves vertically at a different rate than the scroll, creating a parallax depth effect. Uses scroll(root) timeline.

.ease-scroll-parallax

Zoom

Scales up as the element scrolls into view — great for hero images or call-to-action sections.

.ease-scroll-zoom
▼ One More Section ▼

Animation Range Variants

Range Control

Control when the animation plays using range modifiers: quick, full, or exit.

⏱️

Quick

Animation completes within the first 50% of entry.

.ease-range-quick
📏

Full

Animation stretches across the entire viewport coverage.

.ease-range-full
🚪

Exit

Animation triggers when the element scrolls out of view.

.ease-range-exit
