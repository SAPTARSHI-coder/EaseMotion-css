# Animation: Ripple Card Focus

A ripple focus animation for card components. When activated, a soft circular wave expands outward from the center of the card, briefly highlighting it before fading away.

## Features

* **Pure CSS animation** using `@keyframes ripple-focus`
* **Click-triggered interaction**
* **Smooth expanding ripple effect**
* **Accessible focus indication**
* **Responsive design**
* **Reduced motion support**
* **Works well for dashboards, pricing cards, and feature sections**

## Usage

```html
<div class="ripple-card active">
  <h3>Premium Plan</h3>
  <p>Unlimited access to all features.</p>
</div>
```

Apply the `.active` class to trigger the ripple animation. Remove it to reset.
