CSS Top Layer Stacking

Dialogs, popovers, and backdrops — the CSS top layer explained.

What is the Top Layer?

The top layer is a dedicated stacking layer in the browser that renders above all positioned elements. Elements like <dialog>, [popover], and fullscreen content are rendered in the top layer, ensuring they appear above everything else regardless of z-index.

Multiple Top Layer Items

You can have multiple dialogs and popovers open simultaneously. The most recently opened item is on top.

Each gets its own ::backdrop which stacks behind the element but above the page content.
