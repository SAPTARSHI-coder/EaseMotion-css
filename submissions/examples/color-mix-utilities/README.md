# Color Mix Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/color-mix-utilities/style.css">
<div class="color-mix-utilities-card color-mix-utilities-card-featured">Color Mix Utilities Content</div>
```

## Classes
- `.color-mix-utilities-card`: Primary card with breathe animation
- `.color-mix-utilities-card-alt`: Secondary variant
- `.color-mix-utilities-card-ghost`: Ghost/outline card
- `.color-mix-utilities-card-glow`: Glow effect card
- `.color-mix-utilities-card-featured`: Gradient featured card
- `.color-mix-utilities-card-sm / -md / -lg`: Size variants
- `.color-mix-utilities-v-1 through -5`: Opacity variants
- `.color-mix-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.color-mix-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 243deg
- Border radius: 7px / 10px / 25px
