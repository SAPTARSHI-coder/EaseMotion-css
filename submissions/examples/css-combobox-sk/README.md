# CSS Combobox

Dropdown select using `<details>`/`<summary>` with styled option menu.

## Usage
```html
<div class="combobox">
  <details class="combobox__dropdown">
    <summary class="combobox__trigger">Select</summary>
    <div class="combobox__option">Option</div>
  </details>
</div>
```

## Why EaseMotion CSS?
CSS-only dropdown pattern using the details disclosure element.
