# Animated Search & Filter

Search input that starts as an icon-only button and expands into a full input on focus/click. Includes a filter toggle button that opens a CSS-only dropdown panel with checkboxes.

Created by @saurabhhhcodes

## Features
- Expandable search input: icon → full field on focus
- Filter button with CSS-only dropdown using checkbox hack
- Checkbox filter options inside dropdown
- Smooth transitions on expand/collapse
- BEM naming convention
