# Scrollbar Width Utilities

CSS utility classes for the `scrollbar-width` property.

## Usage

```html
<div class="scrollbar-thin">...</div>
```

## Classes

- `.scrollbar-thin` — scrollbar-width: thin;
- `.scrollbar-none` — scrollbar-width: none;
- `.scrollbar-auto` — scrollbar-width: auto;
- `.scrollbar-thin-hover` — scrollbar-width: thin;
- `.scrollbar-none-hover` — scrollbar-width: none;
- `.scrollbar-width-thin` — scrollbar-width: thin;
- `.scrollbar-width-none` — scrollbar-width: none;
- `.scrollbar-width-auto` — scrollbar-width: auto;
- `.scrollbar-w-thin` — scrollbar-width: thin;
- `.scrollbar-w-none` — scrollbar-width: none;
- `.scrollbar-w-auto` — scrollbar-width: auto;
- `.scrollbar-style-thin` — scrollbar-width: thin;
- `.scrollbar-style-none` — scrollbar-width: none;
- `.scrollbar-style-auto` — scrollbar-width: auto;
- `.scrollbar-width-inherit` — scrollbar-width: inherit;
- `.scrollbar-width-initial` — scrollbar-width: initial;
- `.scrollbar-width-unset` — scrollbar-width: unset;
- `.scrollbar-w-inherit` — scrollbar-width: inherit;
- `.scrollbar-w-initial` — scrollbar-width: initial;
- `.scrollbar-w-unset` — scrollbar-width: unset;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
