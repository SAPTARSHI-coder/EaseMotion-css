# Sign Making Company

A modern and responsive **Sign Making Company** landing page built using **EaseMotion CSS** animations.

## Features

* Responsive Hero Section
* Products Grid
* Design Services
* Industries Served
* Portfolio Gallery
* Free Quote Form
* Customer Testimonials
* Footer with Contact Information

## Technologies Used

* HTML5
* CSS3
* EaseMotion CSS

## EaseMotion Classes Used

* `fade-in`
* `fade-up`
* `slide-left`
* `slide-right`
* `zoom-in`
* `hover-lift`
* `hover-grow`
