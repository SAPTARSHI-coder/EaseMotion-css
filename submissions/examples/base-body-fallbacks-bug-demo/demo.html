Bug: body var() without fallbacks

Without the EaseMotion variables layer, the body below would render with transparent background and invisible text (var() resolves to initial).

This paragraph uses the fallback values directly.

  font-size: var(--ease-text-base, 1rem);
  line-height: var(--ease-leading-normal, 1.6);

  background-color: var(--ease-color-bg, #f8fafc);
