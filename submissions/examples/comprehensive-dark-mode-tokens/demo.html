Comprehensive Dark Mode Tokens

This demo showcases how redefining the full neutral palette and lightening semantic colors in dark mode allows for truly seamless theme switching. Try toggling data-theme="dark" to data-theme="light" on the html tag!

Neutral Background

Using .ease-bg-neutral (which maps to neutral-100). In light mode this is very light gray, but in dark mode it intelligently flips to a dark panel color.

Inverted Neutral Card

This card's background uses neutral-800. Thanks to the token inversion, it is dark in light mode, and light in dark mode!
