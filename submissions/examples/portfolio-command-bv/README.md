# Professional Portfolio Command Center

An ultra-premium developer showcase and portfolio center displaying stats cards, skill indices, project showcases, GitHub activity heatmaps, and timeline logs. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Branding Header**: Profile avatar blocks detailing job titles, availability status, and online indicator tags.
- **Hero Statement**: Large introductory statements featuring primary-to-accent text gradients.
- **Achievement Dashboard**: Cards indicating completed projects, years of expertise, client sizes, and awards.
- **Skill Intelligence Scale**: Progressive skill indicators mapping expertise ranges.
- **Project Cards Matrix**: Grid displaying flagship projects with tech stack tags and outlines.
- **GitHub Contribution Heatmap**: Styled grid displaying contribution levels.
- **Experience Timeline**: Career timeline listing positions, companies, dates, and achievements.

## 🎨 Theme Details

- **Palette**: Midnight Space (`#0c0c1d`), Surface Purple (`#12122b`), Accent Cyan (`#22d3ee`), Highlight Coral (`#fb923c`), Text Grey (`#f3f4f6`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft entrance slide-ups, typewriter header sequences, and card hover scaling.
