# Responsive Breakpoint Tester Component

## What does this do?

Provides a visual demonstration of common responsive design breakpoints including Mobile, Tablet, Laptop, and Desktop layouts.

## How is it used?

```html
<div class="breakpoint-card mobile">
    <h3>Mobile</h3>
    <p>&lt; 640px</p>
</div>
```

## Why is it useful?

Responsive design is a fundamental part of modern web development. This component helps developers understand breakpoint ranges and visualize layout adaptations using pure HTML and CSS.
