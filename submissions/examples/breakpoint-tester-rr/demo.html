01

What are Breakpoints?

A breakpoint is a specific viewport width at which your CSS layout switches to a different configuration. Think of breakpoints as the seams where your design intentionally changes shape — not because it breaks, but because a narrower or wider canvas demands a different arrangement of content.

EaseMotion CSS ships with four opinionated default breakpoints — mobile, tablet, laptop, and desktop — designed to cover the vast majority of real-world devices with minimal cognitive overhead.

02

Why Media Queries Matter

Media queries are the CSS mechanism that makes breakpoints real. They let you attach styles conditionally, scoped to a viewport range. Without them, every user — whether on a 320px phone or a 4K monitor — would see the same rigid layout.

CSS Example 
/* Mobile first — base styles apply to all */
.grid { display: grid; grid-template-columns: 1fr; }

/* Tablet and up */
@media (min-width: 768px) {
  .grid { grid-template-columns: 1fr 1fr; }
}

/* Desktop */
@media (min-width: 1440px) {
  .grid { grid-template-columns: repeat(4, 1fr); }
}
03

How Layouts Adapt

Adaptation is not just about columns. As the viewport grows, a well-designed responsive system adjusts multiple layers simultaneously:

  • → Grid columns expand from 1 to 4
  • → Typography scales with fluid clamp()
  • → Navigation shifts from hamburger to inline
  • → Images switch from stacked to side-by-side
  • → Spacing increases proportionally with room
  • → Sidebars appear when horizontal space allows
Usage in EaseMotion CSS

Drop it in. Ship it fast.

EaseMotion CSS ships breakpoint utilities as scoped custom properties and semantic class prefixes. No configuration required — resize and react.

1
Link the stylesheet Add easemotion.css to your <head>
2
Apply responsive classes Use .em-col-1, .em-md:col-2, .em-lg:col-4
3
Resize and verify Drag your browser window — layouts adapt automatically
index.html 
<!-- EaseMotion CSS Responsive Grid -->
<link rel="stylesheet"
      href="easemotion.css">

<div class="em-grid
         em-col-1
