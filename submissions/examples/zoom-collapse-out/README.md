# Zoom Collapse Out

A fast dramatic exit animation where the element shrinks to a point while fading out using pure CSS.

## Features

- Scale-down collapse effect
- Smooth opacity fade
- Center-origin animation
- Fast dramatic exit
- Pure CSS implementation

## Usage

```html
<div class="zoom-box">
  COLLAPSE
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable exit animation
- Pure CSS motion utility
