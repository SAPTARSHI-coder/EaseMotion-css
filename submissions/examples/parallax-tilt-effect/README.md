# Parallax Tilt Effect

## 1. What does this do?

This effect creates a subtle 3D tilt interaction that gives cards and UI elements a sense of depth when users hover over them.

## 2. How is it used?

Apply the tilt class to a card or container element.

```html
<div class="tilt-card">
  <h3>Project Card</h3>
  <p>Hover to see the parallax tilt effect.</p>
</div>
```

## 3. Why is it useful?

Parallax tilt effects make interfaces feel more interactive and engaging by adding depth and motion without JavaScript. They are commonly used in portfolios, product showcases, dashboards, and modern landing pages to draw attention to important content while maintaining a lightweight, CSS-only implementation.
