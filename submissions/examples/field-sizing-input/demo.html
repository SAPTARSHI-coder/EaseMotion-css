field-sizing: content

Inputs and textareas that grow with their content — zero JavaScript

Single-line input

Type — it expands as you type, shrinks as you delete.

Textarea

Auto-grows vertically. No scrollbars inside.

Select elements

Selects can also use field-sizing.

How it works

field-sizing: content makes form controls size to their content, like width: fit-content but for interactive elements. Chrome 123+ supports it.
