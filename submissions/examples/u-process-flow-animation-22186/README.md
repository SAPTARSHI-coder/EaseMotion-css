# ease-process-flow-animation

Animated multi-step workflow component built with pure HTML and CSS.

## Features

- Pure CSS
- Responsive layout
- Animated connectors
- Hover interactions
- Active step animation
- Accessibility support
- prefers-reduced-motion support

- demo.html
- style.css
