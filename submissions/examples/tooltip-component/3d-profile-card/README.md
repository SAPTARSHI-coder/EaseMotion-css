# Cyberpunk Neon Profile Card

A futuristic animated profile card built using pure HTML and CSS.

## Features

- Animated neon gradient border
- Floating card animation
- Glassmorphism effect
- Hover scaling
- Neon glow buttons
- Responsive layout
- Pure CSS

## Files

- demo.html
- style.css

## Usage

Open demo.html in any browser.
