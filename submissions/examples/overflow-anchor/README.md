# Overflow Anchor

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/overflow-anchor/style.css">
<div class="overflow-anchor-card overflow-anchor-card-featured">Overflow Anchor Content</div>
```

## Classes
- `.overflow-anchor-card`: Primary card with breathe animation
- `.overflow-anchor-card-alt`: Secondary variant
- `.overflow-anchor-card-ghost`: Ghost/outline card
- `.overflow-anchor-card-glow`: Glow effect card
- `.overflow-anchor-card-featured`: Gradient featured card
- `.overflow-anchor-card-sm / -md / -lg`: Size variants
- `.overflow-anchor-v-1 through -5`: Opacity variants
- `.overflow-anchor-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.overflow-anchor-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 270deg
- Border radius: 5px / 14px / 18px
