# Cinematic Perspective Text Reveal

## 1. What does this do?
A modern, scroll-driven entrance animation that utilizes the native CSS `view()` timeline to trigger a 3D perspective rotation and slide effect as text enters the viewport.

## 2. How is it used?
```html
<section class="ease-cinematic-wrapper">
  <h1 class="ease-reveal-text">Your Text</h1>
</section>
```
### 3. Why is it useful?
It demonstrates the bleeding edge of CSS browser capabilities without requiring heavy JS scroll-libraries. It allows developers to achieve "Awwwards-level" typography animations with nothing but standard CSS classes.
