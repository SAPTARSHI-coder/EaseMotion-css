# Backdrop Filter Contrast

## Overview
Grid cell utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/backdrop-filter-contrast/style.css">
<div class="backdrop-filter-contrast-grid-wrap backdrop-filter-contrast-cols-3 backdrop-filter-contrast-gap-2">
  <div class="backdrop-filter-contrast-cell">Backdrop Filter Contrast Cell</div>
</div>
```

## Classes
- `.backdrop-filter-contrast-cell`: Primary grid cell
- `.backdrop-filter-contrast-cell-alt`: Secondary variant
- `.backdrop-filter-contrast-cell-accent`: Accent variant
- `.backdrop-filter-contrast-cell-featured`: Gradient featured cell
- `.backdrop-filter-contrast-cell-outline`: Outline variant
- `.backdrop-filter-contrast-cell-sm / -md / -lg / -xl`: Size variants
- `.backdrop-filter-contrast-cols-2 / -3 / -4`: Column count
- `.backdrop-filter-contrast-gap-1 / -2 / -3`: Gap variants
- `.backdrop-filter-contrast-rwd-1 through -3`: Responsive padding
