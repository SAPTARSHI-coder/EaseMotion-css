# Backdrop Filter Contrast

## Overview
Implements `backdrop-filter-contrast` CSS card utilities for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/backdrop-filter-contrast/style.css">
<div class="backdrop-filter-contrast-card-pri">Card Content</div>
```

## Variants
- `.backdrop-filter-contrast-card-pri`: Primary card with glow shadow
- `.backdrop-filter-contrast-card-sec`: Secondary color variant
- `.backdrop-filter-contrast-card-ghost`: Outline/ghost variant
- `.backdrop-filter-contrast-card-glow`: Glow effect card
- `.backdrop-filter-contrast-card-succ / -warn`: State cards
- `.backdrop-filter-contrast-card-sm / -md / -lg`: Size variants
- `.backdrop-filter-contrast-card-hov`: Hover lift effect
- `.backdrop-filter-contrast-shd-1 through -4`: Shadow depth variants
- `.backdrop-filter-contrast-var-1 through -11`: Opacity/shadow variants

## Design Tokens Used
- `--ease-color-primary`: #6c63ff
- `--ease-color-secondary`: #8b5cf6
- `--ease-speed-medium`: 300ms
- `--ease-ease`: cubic-bezier(0.4, 0, 0.2, 1)
