# 3D Parallax Tilt Card

## What it does

Cards that tilt in 3D space following mouse movement, with cursor-following glow effect.

## How to use it

<div class="ease-tilt-card" data-tilt-max="15">
    <div class="ease-tilt-card-inner">
        Content here
        <div class="ease-tilt-glow"></div>
    </div>
</div>

## Variants

- data-tilt-max - Max tilt angle (default 15)
- ease-tilt-reverse - Opposite tilt direction
- ease-tilt-glow - Enhanced neon glow
- ease-tilt-flat - No 3D, only light effect

## Features

- Mouse-following 3D rotation
- Cursor-tracking glow
- Adjustable tilt sensitivity
- Smooth transitions

## Why it fits EaseMotion CSS

- Pure CSS + minimal JS
- 3D transform effects
- Interactive mouse tracking
- Premium visual feedback
