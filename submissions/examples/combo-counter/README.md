# Combo Counter Animation

## What does this do?
A combo counter display with a scaling pop effect and rotating background flare.

## How is it used?
Add `.combo` container with `.count`, `.label`, and `.flare` elements.

## Why is it useful?
Combo counters drive engagement in action games by showing real-time scoring feedback.
