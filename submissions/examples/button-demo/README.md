# 🎨 EaseMotion CSS

A lightweight, composable, and accessibility-first CSS component library designed to create smooth, modern, and consistent UI experiences.

---

## ✨ Overview

EaseMotion CSS provides reusable UI components with a strong focus on:

- ♿ Accessibility (keyboard + screen reader friendly)
- 🎯 Consistent design system
- ⚡ Smooth and meaningful animations
- 🧩 Composable utility-based architecture
- 📱 Responsive and modern UI patterns

---

## 🚀 Features

- Accessible button system with multiple states
- Smooth hover, active, and focus interactions
- Reduced motion support (`prefers-reduced-motion`)
- Fully customizable via CSS variables
- Modular component structure
- Lightweight and dependency-free
