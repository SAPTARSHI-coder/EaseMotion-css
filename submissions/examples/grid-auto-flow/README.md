# Grid Auto Flow

## Overview
Implements `grid-auto-flow` CSS card utilities for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/grid-auto-flow/style.css">
<div class="grid-auto-flow-card-pri">Card Content</div>
```

## Variants
- `.grid-auto-flow-card-pri`: Primary card with glow shadow
- `.grid-auto-flow-card-sec`: Secondary color variant
- `.grid-auto-flow-card-ghost`: Outline/ghost variant
- `.grid-auto-flow-card-glow`: Glow effect card
- `.grid-auto-flow-card-succ / -warn`: State cards
- `.grid-auto-flow-card-sm / -md / -lg`: Size variants
- `.grid-auto-flow-card-hov`: Hover lift effect
- `.grid-auto-flow-shd-1 through -4`: Shadow depth variants
- `.grid-auto-flow-var-1 through -11`: Opacity/shadow variants

## Design Tokens Used
- `--ease-color-primary`: #6c63ff
- `--ease-color-secondary`: #8b5cf6
- `--ease-speed-medium`: 300ms
- `--ease-ease`: cubic-bezier(0.4, 0, 0.2, 1)
