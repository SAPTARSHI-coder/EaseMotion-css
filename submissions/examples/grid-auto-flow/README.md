# Grid Auto Flow

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/grid-auto-flow/style.css">
<li class="grid-auto-flow-row grid-auto-flow-row-active">
  Grid Auto Flow item <span class="grid-auto-flow-tag">Grid Auto Flow</span>
</li>
```

## Classes
- `.grid-auto-flow-row`: List row with hover
- `.grid-auto-flow-row-active`: Active row with slide-in animation
- `.grid-auto-flow-tag / -alt / -outline / -success / -warn`: Tag variants
- `.grid-auto-flow-chip / -primary / -success`: Chip variants
- `.grid-auto-flow-hl`: Highlight text
- `.grid-auto-flow-prio-high / -med / -low`: Priority indicators
- `.grid-auto-flow-idx-1 through -3`: Index numbering styles
