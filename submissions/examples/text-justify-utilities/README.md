# Text Justify Utilities

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/text-justify-utilities/style.css">
<li class="text-justify-utilities-row text-justify-utilities-row-active">
  Text Justify Utilities item <span class="text-justify-utilities-tag">Text Justify Utilities</span>
</li>
```

## Classes
- `.text-justify-utilities-row`: List row with hover
- `.text-justify-utilities-row-active`: Active row with slide-in animation
- `.text-justify-utilities-tag / -alt / -outline / -success / -warn`: Tag variants
- `.text-justify-utilities-chip / -primary / -success`: Chip variants
- `.text-justify-utilities-hl`: Highlight text
- `.text-justify-utilities-prio-high / -med / -low`: Priority indicators
- `.text-justify-utilities-idx-1 through -3`: Index numbering styles
