# Position All Utilities

CSS utility classes for the `position-all` property.

## Usage

```html
<div class="top-auto">...</div>
```

## Classes

- `.top-auto` тАФ top: auto;
- `.top-0` тАФ top: 0;
- `.top-px` тАФ top: 1px;
- `.top-1` тАФ top: 0.25rem;
- `.top-2` тАФ top: 0.5rem;
- `.top-4` тАФ top: 1rem;
- `.top-8` тАФ top: 2rem;
- `.top-full` тАФ top: 100%;
- `.bottom-auto` тАФ bottom: auto;
- `.bottom-0` тАФ bottom: 0;
- `.bottom-1` тАФ bottom: 0.25rem;
- `.bottom-2` тАФ bottom: 0.5rem;
- `.bottom-4` тАФ bottom: 1rem;
- `.bottom-full` тАФ bottom: 100%;
- `.left-auto` тАФ left: auto;
- `.left-0` тАФ left: 0;
- `.left-1` тАФ left: 0.25rem;
- `.left-2` тАФ left: 0.5rem;
- `.left-4` тАФ left: 1rem;
- `.left-full` тАФ left: 100%;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
