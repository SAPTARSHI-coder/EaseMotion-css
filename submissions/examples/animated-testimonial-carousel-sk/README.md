# Animated Testimonial Carousel

A CSS-only testimonial carousel/slider with radio button navigation. Features 5 slides with avatar, quote, name, and title. Slide animation uses opacity + translate with smooth cubic-bezier transitions. Dot indicators with active state highlighting.

**Created by @saurabhhhcodes**

## Features

- Pure CSS — no JavaScript required
- Radio button hack for slide navigation
- 5 testimonial slides with avatar SVGs
- Smooth opacity + translateX slide transitions
- Dot indicator navigation with active highlight
- Glassmorphism container with backdrop blur
- Hover effect on avatar images
- Responsive design for mobile
- Accessible interactive controls

## Usage

Add to your HTML:

```html
<link rel="stylesheet" href="style.css">
```

The carousel uses hidden radio inputs and CSS `:checked` selectors to control which slide is visible. Each slide fades and slides in simultaneously.
