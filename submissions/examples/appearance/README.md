# Appearance Utilities

CSS utility classes for the `appearance` property.

## Usage

```html
<div class="appearance-none">...</div>
```

## Classes

- `.appearance-none` ‚Äî appearance: none;
- `.appearance-auto` ‚Äî appearance: auto;
- `.appearance-button` ‚Äî appearance: button;
- `.appearance-checkbox` ‚Äî appearance: checkbox;
- `.appearance-radio` ‚Äî appearance: radio;
- `.appearance-textfield` ‚Äî appearance: textfield;
- `.appearance-searchfield` ‚Äî appearance: searchfield;
- `.appearance-textarea` ‚Äî appearance: textarea;
- `.appearance-menulist` ‚Äî appearance: menulist;
- `.appearance-listbox` ‚Äî appearance: listbox;
- `.appearance-meter` ‚Äî appearance: meter;
- `.appearance-progress` ‚Äî appearance: progress-bar;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
