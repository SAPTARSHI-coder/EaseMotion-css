# Avatar Stack

Overlapping avatar stack for team lists.

## Usage

```html
<div class="avatar-stack"><span class="avatar-stack__item" style="--i:0">A</span><span class="avatar-stack__item" style="--i:1">B</span><span class="avatar-stack__item" style="--i:2">C</span><span class="avatar-stack__more">+4</span></div>
```

## Why EaseMotion CSS

Pure CSS animation utility — no JavaScript required.
