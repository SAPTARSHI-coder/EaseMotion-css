# Numeric Stepper

## What does this do?
Adds `.ease-stepper` — a numeric input with increment/decrement buttons.

```html
<div class="ease-stepper">
  <button class="ease-stepper-dec">-</button>
  <input type="number" value="1">
  <button class="ease-stepper-inc">+</button>
</div>
```

## Preview
Open `demo.html` in your browser.
