# Quest Marker Bounce Animation

## What does this do?
A bouncing quest marker icon with shadow ring and expanding pulse rings.

## How is it used?
Add `.marker` container with `.icon`, `.ring`, and `.pulse` elements.

## Why is it useful?
Quest markers guide players in game UIs and navigation systems.
