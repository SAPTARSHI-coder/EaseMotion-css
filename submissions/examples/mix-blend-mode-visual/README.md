# Mix Blend Mode Visual

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/mix-blend-mode-visual/style.css">
<li class="mix-blend-mode-visual-row mix-blend-mode-visual-row-active">
  Mix Blend Mode Visual item <span class="mix-blend-mode-visual-tag">Mix Blend Mode Visual</span>
</li>
```

## Classes
- `.mix-blend-mode-visual-row`: List row with hover
- `.mix-blend-mode-visual-row-active`: Active row with slide-in animation
- `.mix-blend-mode-visual-tag / -alt / -outline / -success / -warn`: Tag variants
- `.mix-blend-mode-visual-chip / -primary / -success`: Chip variants
- `.mix-blend-mode-visual-hl`: Highlight text
- `.mix-blend-mode-visual-prio-high / -med / -low`: Priority indicators
- `.mix-blend-mode-visual-idx-1 through -3`: Index numbering styles
