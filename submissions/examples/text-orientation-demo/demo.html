Text-Combine-Upright

text-combine-upright compresses digits and short text runs into a single upright character space within vertical text. Compare the dates below — with and without the property.

Without text-combine-upright

Digits are rotated sideways, each taking one character width.

Property Reference

  • writing-mode: vertical-rl — Text flows top-to-bottom, right-to-left. Traditional CJK layout.
  • writing-mode: vertical-lr — Text flows top-to-bottom, left-to-right. Less common but useful for some Mongolian scripts.
  • text-orientation: mixed — CJK upright, Latin rotated 90° (default in vertical writing).
  • text-orientation: upright — All characters rendered upright. Latin becomes hard to read.
  • text-orientation: sideways — All characters rotated 90°. Good for all-Latin vertical headings.
  • text-combine-upright: all — Compress consecutive digits into a single glyph space.
