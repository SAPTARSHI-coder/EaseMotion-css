# Animated Fitness Progress Dashboard

A responsive fitness dashboard featuring activity tracking, workout progress indicators, and achievement badges.

## Features

- Daily activity cards
- Animated progress bars
- Achievement badges
- Responsive layout
- Hover interactions
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more fitness metrics
- Change progress percentages
- Update badge styles
- Customize colors and spacing
