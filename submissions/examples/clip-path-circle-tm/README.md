`clip-path-circle-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/clip-path-circle-tm/style.css">
<div class="clip-path-circle-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .clip-path-circle: creates perfect circle

## Impact

Non-border-radius circular masking for images
