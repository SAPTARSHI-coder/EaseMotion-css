# Searchable List

## What does this do?
Adds `.ease-searchable` — a list with a search input that filters and highlights matches.

```html
<div class="ease-searchable">
  <input type="search" class="ease-searchable-input">
  <ul class="ease-searchable-list">
    <li>Apple</li>
  </ul>
</div>
```

## Preview
Open `demo.html` in your browser.
