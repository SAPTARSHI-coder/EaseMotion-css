# Power-Up Glow Animation

## What does this do?
A power-up icon with pulsing core and rotating light beams.

## How is it used?
Add `.powerup` container with `.core` and `.beams` elements.

## Why is it useful?
Power-up effects reward players and communicate boosted states in games.
