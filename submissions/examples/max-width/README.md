# Max Width Utilities

CSS utility classes for the `max-width` property.

## Usage

```html
<div class="max-w-0">...</div>
```

## Classes

- `.max-w-0` тАФ max-width: 0;
- `.max-w-none` тАФ max-width: none;
- `.max-w-full` тАФ max-width: 100%;
- `.max-w-screen` тАФ max-width: 100vw;
- `.max-w-min` тАФ max-width: min-content;
- `.max-w-max` тАФ max-width: max-content;
- `.max-w-fit` тАФ max-width: fit-content;
- `.max-w-xs` тАФ max-width: 20rem;
- `.max-w-sm` тАФ max-width: 24rem;
- `.max-w-md` тАФ max-width: 28rem;
- `.max-w-lg` тАФ max-width: 32rem;
- `.max-w-xl` тАФ max-width: 36rem;
- `.max-w-2xl` тАФ max-width: 42rem;
- `.max-w-3xl` тАФ max-width: 48rem;
- `.max-w-4xl` тАФ max-width: 56rem;
- `.max-w-5xl` тАФ max-width: 64rem;
- `.max-w-6xl` тАФ max-width: 72rem;
- `.max-w-7xl` тАФ max-width: 80rem;
- `.max-w-prose` тАФ max-width: 65ch;
- `.max-w-screen-sm` тАФ max-width: 640px;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
