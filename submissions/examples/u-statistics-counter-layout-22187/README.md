# ease-statistics-counter-layout

Animated statistics section with circular indicators and progress bars.

## Features

- Pure HTML & CSS
- Circular progress indicators
- Animated progress bars
- Responsive grid
- Hover animations
- Accessibility support
- prefers-reduced-motion support

## Files

- demo.html
- style.css

## Usage

```html
<section class="stats-section">
...
</section>
```
