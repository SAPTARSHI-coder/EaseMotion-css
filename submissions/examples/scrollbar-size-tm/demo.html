scrollbar-width

Custom Scrollbar Width

The scrollbar-width property lets you control the thickness of scrollbars in Firefox, enabling thin and invisible scrollbar designs without JavaScript.

Three Width Modes
Compare the three scrollbar-width values. Note: this property is Firefox-only; use ::-webkit-scrollbar for Chrome/Safari styling.

scrollbar-width: auto

default width

The default scrollbar uses the platform-native width. On Windows this is typically 16px, on macOS 15px, and on Linux varies by GTK theme.

Auto-width scrollbars are familiar and accessible but take up significant horizontal space in narrow layouts.

Keep scrolling to see the scrollbar. It uses the system default appearance on Firefox.

The auto value is useful when you want to reset a previously overridden scrollbar back to the platform default.

Accessibility note: very thin scrollbars may be harder to interact with for users with motor impairments.

scrollbar-width: thin

narrower, ~8px

The thin scrollbar reduces visual weight from 16px down to roughly 8px, freeing horizontal space in sidebars and compact layouts.

This is ideal for chat interfaces, code editors, and mobile-inspired designs where a slim scrollbar looks cleaner.

Keep scrolling to see the thin scrollbar. It still provides full interaction but with a reduced visual footprint.

The thin variant maintains hover states and drag functionality while appearing more elegant.

Pair with ::-webkit-scrollbar styling in Chromium browsers for cross-browser thin scrollbars.

scrollbar-width: none

invisible scrollbar

Hiding the scrollbar entirely is useful for hidden overflow containers where scrolling is done via keyboard or custom scroll indicators.

The content remains fully scrollable — only the scrollbar track and thumb are hidden.

Keep scrolling to confirm the content is still accessible. Use JavaScript scroll indicators for user feedback.

Accessibility warning: hiding scrollbars can confuse users about whether content is scrollable. Always provide visual or textual cues.

This approach works alongside overflow-y: auto to conditionally show the scrollbar only when needed.
