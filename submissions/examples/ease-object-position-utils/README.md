# ease-object-position utilities

Controls `object-position` for positioning replaced elements (images, videos) within their box.

## Usage

```html
<img src="..." class="ease-object-position-center" alt="" />
```

## Classes

| Class                         | CSS Property              |
| ----------------------------- | ------------------------- |
| `ease-object-position-top`    | `object-position: top`    |
| `ease-object-position-bottom` | `object-position: bottom` |
| `ease-object-position-center` | `object-position: center` |
| `ease-object-position-left`   | `object-position: left`   |
| `ease-object-position-right`  | `object-position: right`  |
