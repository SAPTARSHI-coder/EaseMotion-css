2. SCSS Architecture Reference

Here is how the variables are defined and overridden using the Sass module system:

Variables Declaration (scss/_variables.scss) 
// Exposed with default flags to allow modular overrides
$speed-fast:        var(--ease-speed-fast) !default;
$speed-medium:      var(--ease-speed-medium) !default;
$speed-slow:        var(--ease-speed-slow) !default;

$ease-ease:         cubic-bezier(0.4, 0, 0.2, 1) !default;
$ease-bounce:       cubic-bezier(0.34, 1.56, 0.64, 1) !default;
Custom Workspace Overrides (style.scss) 
// Load variables using @use with configuration
@use '../../../scss/variables' with (
  $speed-fast: 150ms,
  $speed-medium: 600ms,
  $speed-slow: 1500ms,
  $ease-bounce: cubic-bezier(0.68, -0.6, 0.32, 1.6)
);

.animate-fast {
  transition: transform variables.$speed-fast variables.$ease-ease;
}
