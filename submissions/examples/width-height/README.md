# Width Height Utilities

CSS utility classes for the `width-height` property.

## Usage

```html
<div class="w-auto">...</div>
```

## Classes

- `.w-auto` ‚Äî width: auto;
- `.w-full` ‚Äî width: 100%;
- `.w-screen` ‚Äî width: 100vw;
- `.w-min` ‚Äî width: min-content;
- `.w-max` ‚Äî width: max-content;
- `.w-fit` ‚Äî width: fit-content;
- `.w-px` ‚Äî width: 1px;
- `.w-1` ‚Äî width: 0.25rem;
- `.w-2` ‚Äî width: 0.5rem;
- `.w-4` ‚Äî width: 1rem;
- `.w-8` ‚Äî width: 2rem;
- `.w-12` ‚Äî width: 3rem;
- `.w-16` ‚Äî width: 4rem;
- `.w-20` ‚Äî width: 5rem;
- `.w-24` ‚Äî width: 6rem;
- `.w-32` ‚Äî width: 8rem;
- `.w-40` ‚Äî width: 10rem;
- `.w-48` ‚Äî width: 12rem;
- `.w-56` ‚Äî width: 14rem;
- `.w-64` ‚Äî width: 16rem;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
