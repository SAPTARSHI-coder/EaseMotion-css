# Counter Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/counter-utilities/style.css">
<div class="counter-utilities-card counter-utilities-card-featured">Counter Utilities Content</div>
```

## Classes
- `.counter-utilities-card`: Primary card with breathe animation
- `.counter-utilities-card-alt`: Secondary variant
- `.counter-utilities-card-ghost`: Ghost/outline card
- `.counter-utilities-card-glow`: Glow effect card
- `.counter-utilities-card-featured`: Gradient featured card
- `.counter-utilities-card-sm / -md / -lg`: Size variants
- `.counter-utilities-v-1 through -5`: Opacity variants
- `.counter-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.counter-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 263deg
- Border radius: 6px / 14px / 19px
