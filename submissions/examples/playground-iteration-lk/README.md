# Animation Playground with Iteration Count

## What does this do?
This is an interactive playground that allows developers to select an animation type, adjust its duration, set a delay, and most importantly, configure its **iteration count** (e.g., `1`, `3`, `infinite`). It then generates the corresponding CSS classes and applies them inline to a preview box so the user can see the result live.

## How is it used?
1. Check the boxes for the animations you want to apply.
2. Edit the duration and delay in milliseconds.
3. Edit the iteration count (e.g., `infinite` or `3`).
4. Watch the preview box update instantly, and copy the generated `ease-` classes from the bottom panel.

## Why is it useful?
Users must often manually edit code locally to experiment with combinations of animations, especially when controlling how many times an animation should loop (which modifies CSS custom properties). This playground provides a quick, visual sandbox to test these combinations in real time without writing any code.
