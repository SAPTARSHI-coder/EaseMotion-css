# Neon Text Shadow

### What does this do?
Demonstrates neon glow effects using layered `text-shadow` declarations. Four neon colors are featured — Blue, Pink, Green, and Yellow — each with 3–5 progressively larger shadow layers to simulate a realistic glow. Every color includes a pulsing variant that oscillates the glow intensity.

### How is it used?
Open `demo.html` in any browser. Apply classes like `neon-blue`, `neon-pink`, etc. to any text element. Add the `pulse` class for an animated breathing glow. The `text-shadow` layers are fully customizable — adjust the color, blur radius, or spread values in the CSS to tweak the intensity.

### Why is it useful?
Neon text effects are a staple in gaming, entertainment, and creative brand interfaces. This example provides a pure-CSS approach (no canvas, no JS, no external assets) that is performant, accessible, and easy to drop into any project. The pulsing animation respects `prefers-reduced-motion` for accessibility compliance.
