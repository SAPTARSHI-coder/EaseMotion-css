# Threat Intelligence Command Platform

An enterprise-grade threat intelligence command platform featuring indicator-of-compromise trackers, actor profiling tables, dynamic risk scorecards, and live threat feed console logs. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Reputation Header**: Brand logo, system active indicators, and global threat status ratings.
- **Risk Scorecard Cards**: Score values, target limits, and tracked IOC metrics.
- **Actor Matrix Table**: Table mapping origin locations, campaigns, threat groups, and risk points.
- **Sync Actions Console**: Control buttons triggering active threat database synchronization scripts.
- **Log Telemetry Shell**: Console logs streaming vulnerability updates.

## 🎨 Theme Details

- **Palette**: Space Navy (`#0b0f19`), Surface Grey (`#161f30`), Accent Violet (`#8b5cf6`), Threat Rose (`#f43f5e`), Status Green (`#10b981`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Blinking alerts, warning pulses, and transition cards.
