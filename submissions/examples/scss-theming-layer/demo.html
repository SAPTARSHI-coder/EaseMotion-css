2. SCSS Theming Integration Guide

Here is how the theming system is defined inside the framework and how you can implement custom themes using maps and mixins:

Theme Map & Function Definition (scss/_theme.scss) 
@use "sass:map";

$theme-colors: (
  "primary":    var(--ease-color-primary, #6366f1),
  "background": var(--ease-color-bg, #f8fafc),
  "surface":    var(--ease-color-surface, #ffffff)
) !default;

@function color($name) {
  @return map.get($theme-colors, $name);
}

@mixin theme-provider($color-map) {
  @each $key, $value in $color-map {
    --ease-color-#{$key}: #{$value};
  }
}
Writing Theme Variations (style.scss) 
@use '../../../scss/theme';

$dark-theme: (
  "primary":    #818cf8,
  "background": #0f172a,
  "surface":    #1e293b
);

.theme-dark {
  @include theme.theme-provider($dark-theme);
}

body {
  background-color: theme.color("background");
  color: theme.color("text");
}
