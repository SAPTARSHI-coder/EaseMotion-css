# Quotes Css: Custom Quotation Mark Styling

Demonstrates CSS quotes property for custom quotation marks in multiple languages, blockquote styling with open/close quotes, and nested quotation levels using EaseMotion design tokens.

## Features

- Demonstrates **quotes** CSS property with multiple practical variants
- Uses EaseMotion design tokens (`--ease-color-*`, `--ease-space-*`, `--ease-radius-*`)
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`
- Multiple size/style variants for real-world usage

## Usage

Apply the utility class to any element:

```html
<div class="quotes-default">Content</div>
```

## Why is it useful?

The **quotes** CSS property is part of the modern CSS specification and enables
fine-grained control over quotes-css. It integrates seamlessly with EaseMotion's design token
system, making it easy to maintain consistent spacing, colors, and typography across your project.
