✧ transition-behavior: allow-discrete

Animating display and overlay — smooth open/close transitions

transition-behavior: allow-discrete enables CSS transitions on discrete properties like display and overlay, which previously could not be animated. Combined with @starting-style, this enables smooth open/close animations.

Multiple Properties

Animating display + opacity + transform + overlay simultaneously

Multi-Property

Four properties animating at once: display, overlay, opacity, and transform.

  • display: none → block
  • overlay: none → auto
  • opacity: 0 → 1
  • transform: scale(0.9) → scale(1)

Key CSS

.element {
  transition:
    display 0.4s allow-discrete,
    overlay 0.4s allow-discrete,
    opacity 0.4s,
    transform 0.4s;
}

.element.hidden {
  display: none;
  opacity: 0;
  transform: scale(0.9);
}

@starting-style {
  .element:not(.hidden) {
    opacity: 0;
    transform: scale(0.9);
  }
}
