Developer Integration

1. CSS — Automatic Detection (zero config)

/* Already included in easemotion.css */ @media (prefers-reduced-motion: reduce) { :root { --ease-speed-fast: 0.01ms; --ease-speed-medium: 0.01ms; --ease-speed-slow: 0.01ms; --ease-animation-iterations: 1; } }

2. JavaScript — Runtime Detection

// Detect system preference const mq = window.matchMedia('(prefers-reduced-motion: reduce)'); if (mq.matches) { // User prefers reduced motion document.documentElement.dataset.reduceMotion = 'true'; } // Listen for changes mq.addEventListener('change', (e) => { document.documentElement.dataset.reduceMotion = e.matches; });

3. In-Page Toggle (no OS change required)

<!-- Add a toggle button --> <button onclick="toggleMotion()">Toggle Motion</button> <script> function toggleMotion() { const html = document.documentElement; const current = html.dataset.reduceMotion === 'true'; html.dataset.reduceMotion = !current; } </script>
