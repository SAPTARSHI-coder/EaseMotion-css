# Flex Direction Utilities

CSS utility classes for the `flex-direction` property.

## Usage

```html
<div class="flex-row">...</div>
```

## Classes

- `.flex-row` тАФ flex-direction: row;
- `.flex-row-reverse` тАФ flex-direction: row-reverse;
- `.flex-col` тАФ flex-direction: column;
- `.flex-col-reverse` тАФ flex-direction: column-reverse;
- `.flex-grow-0` тАФ flex-grow: 0;
- `.flex-grow` тАФ flex-grow: 1;
- `.flex-grow-2` тАФ flex-grow: 2;
- `.flex-grow-3` тАФ flex-grow: 3;
- `.flex-shrink-0` тАФ flex-shrink: 0;
- `.flex-shrink` тАФ flex-shrink: 1;
- `.flex-shrink-2` тАФ flex-shrink: 2;
- `.flex-basis-auto` тАФ flex-basis: auto;
- `.flex-basis-0` тАФ flex-basis: 0;
- `.flex-basis-full` тАФ flex-basis: 100%;
- `.flex-basis-half` тАФ flex-basis: 50%;
- `.flex-basis-quarter` тАФ flex-basis: 25%;
- `.flex-basis-third` тАФ flex-basis: 33.333%;
- `.flex-auto` тАФ flex: 1 1 auto;
- `.flex-none` тАФ flex: none;
- `.flex-initial` тАФ flex: 0 1 auto;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
