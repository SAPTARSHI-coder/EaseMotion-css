# animated-stat-card-sk

Dashboard stat cards grid (4 cards). Each card shows an icon, animated value, label, and trend indicator with up/down styling. The values use a CSS count-up entrance animation. Cards lift on hover with an enhanced shadow. Each card is color-coded via a left border and matching icon colour.

**Features:**
- Responsive auto-fit grid
- Count-up entrance animation
- Hover lift effect with shadow
- Color-coded per card (revenue, users, orders, conversion)
- Trend badges (up green, down red)
