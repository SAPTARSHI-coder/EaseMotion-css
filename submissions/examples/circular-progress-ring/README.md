# Circular Progress Ring

A radial progress ring component using CSS conic-gradient — pure CSS.

## Features
- Custom percentage via `--pct` property
- Donut shape with center cutout
- Progress color gradient fill

## Usage
```html
<div class="ring" style="--pct:75"><span>75%</span></div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
