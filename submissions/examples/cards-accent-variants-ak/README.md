# Cards: Missing Accent Variants Demo

Submission for issue [#5946](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/5946)

## What this demonstrates

The EaseMotion CSS card component is missing `accent-info` and `accent-secondary` color variants. This demo shows what those variants would look like alongside the existing primary/success/warning/danger accent variants.

## Files

- `demo.html` — Demonstrates all card accent variants including the missing info and secondary
- `style.css` — Card styles with all accent variant classes
