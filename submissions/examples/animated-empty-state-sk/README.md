# Animated Empty State

CSS-only empty state component with icon, message, and call-to-action.

## Features

- Four variants: no results, no data, error, empty cart
- Distinct icon colors for each variant (indigo, green, red, orange)
- Subtle icon pulse animation
- Card hover lift effect
- Centered layout with fade-in entrance
- Primary and secondary CTA button styles
- Grid layout responsive down to single column

## Usage

Include the `style.css` and use the BEM class structure. Each empty state card uses `emptystate__icon--{variant}` for color theming. Add `emptystate__btn--primary` for the main action button.

## Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). Uses CSS grid, animations, custom properties, and transforms.
