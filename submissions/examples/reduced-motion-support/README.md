# Reduced Motion Accessibility Support

This feature demonstrates how to use the `prefers-reduced-motion` media query to improve accessibility for motion-sensitive users.

## Features

- Animated motion box
- Automatic animation disabling for reduced motion users
- Accessibility-friendly behavior

## Usage

Enable reduced motion in your operating system accessibility settings to see the effect.

## Files

- `demo.html` → Demo page
- `style.css` → Animation and accessibility styles
