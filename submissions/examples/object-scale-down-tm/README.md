`object-scale-down-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/object-scale-down-tm/style.css">
<div class="object-scale-down-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .object-scale-down: contain if larger than natural size

## Impact

Prevents oversized images from breaking layouts
