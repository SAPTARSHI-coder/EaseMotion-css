# Glass Profile Card

A responsive glassmorphism-inspired profile card built using pure CSS.

## Features

- Frosted glass effect
- Backdrop blur
- Responsive layout
- Hover interaction
- Status badge
- Statistics section
- Social links
- Action buttons
- Mobile-friendly design

## Usage

```html
<div class="ease-glass-card">
  ...
</div>
```

## Main Class

```css
.ease-glass-card
```

## Included Elements

- Profile image
- User information
- Online status indicator
- Statistics grid
- Follow button
- Message button
- Social links

## Why It Fits EaseMotion CSS

- Human-readable naming
- Modern UI component
- Lightweight implementation
- Easy customization
- Smooth interaction effects
- Responsive by default
