Accordion / Collapse Demo

EaseMotion CSS is a human-readable, animation-first CSS framework. It requires zero dependencies and has no build step required, making it incredibly easy to drop into any project.
Just copy the HTML structure and ensure you have included `accordion.css` (or `easemotion.min.css`). Also, include `accordion.js` to handle the click events to toggle the `is-active` class. The CSS Grid transitions will handle the smooth opening and closing animations automatically.
