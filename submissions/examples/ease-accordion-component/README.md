# Accordion / Collapse Component

A clean, modern CSS-only (with minimal JS for toggling) Accordion component that matches the existing framework's design language.

## Features
- Smooth opening and closing animations using CSS Grid transitions.
- Hover and focus states for accessibility.
- Support for CSS variables to easily change colors and padding.
- Zero external dependencies.

## Usage
Include `style.css` and `script.js` in your HTML file, and use the `.ease-accordion` class structure as shown in `demo.html`.
