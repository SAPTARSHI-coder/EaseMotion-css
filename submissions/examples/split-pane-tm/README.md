# Split Pane Resizable

## What does this do?
Adds `.ease-split` — a horizontally resizable two-pane layout.

```html
<div class="ease-split">
  <div class="ease-split-pane">Left</div>
  <div class="ease-split-handle"></div>
  <div class="ease-split-pane">Right</div>
</div>
```

## Preview
Open `demo.html` in your browser.
