# ease-profile-card-flip

A modern 3D profile card flip animation for EaseMotion CSS.

## Features

- Pure CSS
- 3D Perspective
- Front & Back Card
- Hover / Focus Flip
- Responsive Layout
- Accessibility Support
- prefers-reduced-motion support

## Files

- demo.html
- style.css

## Usage

```html
<div class="ease-profile-card-flip">
    ...
</div>
```

## Preview

Hover or focus on the card to reveal additional profile information and social links with a smooth 3D flip animation.
