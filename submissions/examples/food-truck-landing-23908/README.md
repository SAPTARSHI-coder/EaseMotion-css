# Food Truck Landing Page — Rolling Bites

## What is this?

A complete, copy-paste-ready full-page landing page for a street food business built using EaseMotion CSS classes. Features hero section with food truck visual, menu grid, location/schedule, story section with stats, social media grid, and catering CTA.

## EaseMotion CSS Classes Showcased

| Class | Usage |
|-------|-------|
| `ease-animate-fade-in` | Entrance animations |
| `ease-animate-slide-up` | Scroll-triggered reveals |
| `ease-hover-lift` | Card hover effects |
| `ease-glass` | Glassmorphism nav background |
| `ease-gradient-text` | Logo and highlight text |
| Various `ease-color-*` | Theme token usage in custom CSS |

## Sections

1. **Hero** — Bold headline, floating food emojis, animated truck visual, dual CTAs
2. **Menu** — 6-item grid with categories, descriptions, and prices
3. **Story** — Brand origin story with stats (meals served, rating, cities)
4. **Schedule** — 7-day weekly location grid
5. **Social** — 6 social media link cards
6. **Catering CTA** — Prominent call-to-action section
7. **Footer** — Brand name, links, copyright

## Notes

- Single `demo.html` — opens directly in browser
- Fully responsive (mobile, tablet, desktop)
- Realistic placeholder content throughout
- Smooth scroll navigation
- Respects `prefers-reduced-motion`
