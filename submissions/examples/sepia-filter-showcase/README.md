# Sepia Filter Utility Showcase

Demonstrates the implementation of the CSS sepia filter utilities under issue #15203.

## Sepia Utilities Included

- `.ease-filter-sepia`: Applies 100% sepia filter to the target element.
- `.ease-filter-sepia-0`: Removes any active sepia filter (setting it to 0).

## Showcase Layout

Open `demo.html` in a web browser. Hovering over the image card dynamically shifts it from vintage sepia to full color using standard transitions and `.ease-filter-sepia-0`.

Closes #15203
