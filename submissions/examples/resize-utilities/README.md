# Resize Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/resize-utilities/style.css">
<div class="resize-utilities-card resize-utilities-card-featured">Resize Utilities Content</div>
```

## Classes
- `.resize-utilities-card`: Primary card with breathe animation
- `.resize-utilities-card-alt`: Secondary variant
- `.resize-utilities-card-ghost`: Ghost/outline card
- `.resize-utilities-card-glow`: Glow effect card
- `.resize-utilities-card-featured`: Gradient featured card
- `.resize-utilities-card-sm / -md / -lg`: Size variants
- `.resize-utilities-v-1 through -5`: Opacity variants
- `.resize-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.resize-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
