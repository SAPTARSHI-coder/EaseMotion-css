# aspect ratio cards tm

Real CSS utilities using EaseMotion-css design tokens.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/aspect-ratio-cards-tm/style.css">
```

## Tokens Used
- ease-color-primary, ease-color-secondary, ease-color-success, ease-color-info, ease-color-warning
- ease-radius-md, ease-radius-lg, ease-radius-xl
- ease-shadow-sm, ease-shadow-md, ease-shadow-lg
- ease-space-2, ease-space-3, ease-space-4, ease-space-6, ease-space-8
- ease-text-sm, ease-text-base, ease-text-lg, ease-text-xl
- ease-font-sans, ease-ease, ease-ease-bounce, ease-speed-fast, ease-speed-medium, ease-speed-slow
- ease-color-surface, ease-glow-primary, ease-color-neutral-300, ease-color-neutral-900
- Dark mode and prefers-reduced-motion support
