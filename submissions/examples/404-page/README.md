# EaseMotion 404 Page

## 🐛 Description
This submission adds a custom animated **404 Not Found page** for EaseMotion CSS. It provides a clean, modern, and user-friendly error screen when users navigate to a non-existent route.

---

## ✨ Features

- 🎨 Modern glassmorphism card UI
- ⚡ Smooth floating animation effect
- 📱 Fully responsive design
- 🧭 Clear navigation back to home
- 🎯 Consistent with EaseMotion CSS utility classes
- 💡 Minimal and clean implementation
