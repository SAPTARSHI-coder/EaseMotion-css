# Ease Card Stack Hover

A responsive **Ease Card Stack Hover** animation built using **HTML5** and **CSS3**. The component demonstrates stacked cards that fan out with perspective, staggered transforms, depth, and elevation effects when hovered, making it suitable for modern portfolios, dashboards, landing pages, and feature showcases.

---

## Features

- Pure HTML5 & CSS3
- Interactive stacked card layout
- CSS perspective transforms
- Staggered hover animations
- Floating card animation
- Responsive design
- Reusable component
- Smooth transitions
- Accessibility support
- No JavaScript required

## Accessibility

Supports reduced motion using:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.
