# Haven House – Nonprofit Homeless Shelter Landing Page

A complete, ready-to-copy landing page for a homeless shelter nonprofit. Built entirely with **EaseMotion CSS** utility classes – no custom keyframes, no JavaScript (except for a simple count‑up animation on the “housed” stat).

## Sections
- **Header** – clean navigation
- **Hero** – dignity-focused messaging with a donate CTA
- **Services** – three‑card grid (shelter, meals, case management)
- **How to Help** – donate / volunteer / in‑kind cards with call‑to‑action buttons
- **Impact Stats** – large numbers with a count‑up animation for individuals housed
- **Corporate Partnership** – CTA for businesses
- **Newsletter Signup** – email form
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-justify-between`, `ease-items-center`, `ease-gap-*`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-blue-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`, `ease-input`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-blue-700`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct (as used in the template).
3. Open the file in any modern browser – no build step required.
