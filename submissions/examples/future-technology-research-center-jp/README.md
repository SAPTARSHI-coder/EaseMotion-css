# Future Technology Research Center

A futuristic research intelligence dashboard built using pure HTML and CSS.

## Features

- Research Program Analytics
- Lab Experiment Monitoring
- Academic Partnership Metrics
- Research Funding Dashboard
- Publication Tracking System
- Research Timeline Feed
- Glassmorphism Design
- Violet-Pink Theme
- Fully Responsive Layout

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
