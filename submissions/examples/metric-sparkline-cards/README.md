# Metric Sparkline Cards

## What does this do?
A pair of analytics cards with large metric values and animated sparkline-style chart fills.

## How is it used?
```html
<div class="metric-grid">
  <article><span>Revenue</span><strong>42K</strong><i></i></article>
</div>
```

## Why is it useful?
It gives EaseMotion CSS a compact analytics card pattern for dashboards and reporting surfaces.
