Viewport Units: vh vs dvh vs svh vs lvh

Full-height hero sections that work on mobile with dynamic toolbars

100vh

Traditional viewport height — includes the URL bar area. On mobile, the bottom content may be hidden behind the browser chrome.

100dvh

Dynamic viewport height — adjusts as the browser toolbar appears/disappears. Always fits the visible area exactly.

100svh

Smallest viewport height — the minimum possible size. Content stays visible even when the toolbar is shown.

100lvh

Largest viewport height — the maximum possible size when the toolbar is hidden. May be taller than the visible area when chrome is shown.

Fallback Chain

Use this pattern for maximum browser compatibility:

/* Fallback for older browsers */
height: 100vh;
/* Modern dynamic viewport */
height: 100dvh;
/* Smallest viewport safety net */
height: 100svh;

The cascade ensures that newer units override older ones when supported.
