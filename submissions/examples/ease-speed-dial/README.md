# Ease Speed Dial Menu

Floating action button that expands into multiple action buttons with staggered animations.

## Usage

```html
<div class="ease-speed-dial">
    <button class="ease-speed-dial__trigger">+</button>
    <div class="ease-speed-dial__menu">
        <button class="ease-speed-dial__item">Share</button>
        <button class="ease-speed-dial__item">Edit</button>
        <button class="ease-speed-dial__item">Delete</button>
    </div>
</div>
