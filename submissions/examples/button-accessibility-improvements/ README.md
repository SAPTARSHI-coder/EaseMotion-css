# Button Accessibility Improvements

## Description
This demo showcases accessible button components built using EaseMotion CSS principles.

## Features
- Primary and secondary buttons
- Disabled state handling
- Keyboard focus outline support
- Smooth hover transitions

## How to Run
Open `demo.html` in any browser.
