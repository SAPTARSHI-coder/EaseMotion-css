# Tab Size Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/tab-size-utilities/style.css">
<div class="tab-size-utilities-card tab-size-utilities-card-featured">Tab Size Utilities Content</div>
```

## Classes
- `.tab-size-utilities-card`: Primary card with breathe animation
- `.tab-size-utilities-card-alt`: Secondary variant
- `.tab-size-utilities-card-ghost`: Ghost/outline card
- `.tab-size-utilities-card-glow`: Glow effect card
- `.tab-size-utilities-card-featured`: Gradient featured card
- `.tab-size-utilities-card-sm / -md / -lg`: Size variants
- `.tab-size-utilities-v-1 through -5`: Opacity variants
- `.tab-size-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.tab-size-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
