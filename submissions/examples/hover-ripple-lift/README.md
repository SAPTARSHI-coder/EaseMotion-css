# Hover Ripple Lift

## Overview

**Hover Ripple Lift** is a lightweight CSS hover animation that combines two visual effects: a smooth upward lift and an expanding ripple originating from the center of the element. The animation is designed to provide immediate visual feedback, making interactive elements feel more responsive and engaging without relying on JavaScript.

---

## What does this do?

This component creates a hover animation where the element gently lifts upward while a circular ripple expands from its center, giving users a clear visual indication that the element is interactive.

---

## How is it used?

Include `style.css` in your HTML file and apply the custom class to any interactive element.

### Example

```html
<button class="hover-ripple-lift">
    Hover Me
</button>
```

The effect works well with:

* Buttons
* Cards
* Call-to-action sections
* Navigation items
* Interactive icons

## Why is it useful?

Interactive interfaces benefit from subtle visual feedback that helps users understand which elements can be clicked or interacted with. The Hover Ripple Lift animation improves usability by providing a clear hover state while maintaining a clean and modern appearance.

This aligns with EaseMotion CSS's philosophy of creating expressive, reusable, and beginner-friendly motion effects using simple, maintainable CSS that enhances the user experience without unnecessary complexity.
