# List Style Image

## Overview
Grid cell utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/list-style-image/style.css">
<div class="list-style-image-grid-wrap list-style-image-cols-3 list-style-image-gap-2">
  <div class="list-style-image-cell">List Style Image Cell</div>
</div>
```

## Classes
- `.list-style-image-cell`: Primary grid cell
- `.list-style-image-cell-alt`: Secondary variant
- `.list-style-image-cell-accent`: Accent variant
- `.list-style-image-cell-featured`: Gradient featured cell
- `.list-style-image-cell-outline`: Outline variant
- `.list-style-image-cell-sm / -md / -lg / -xl`: Size variants
- `.list-style-image-cols-2 / -3 / -4`: Column count
- `.list-style-image-gap-1 / -2 / -3`: Gap variants
- `.list-style-image-rwd-1 through -3`: Responsive padding
