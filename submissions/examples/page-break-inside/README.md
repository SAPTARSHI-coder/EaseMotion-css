# Page Break Inside

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/page-break-inside/style.css">
<div class="page-break-inside-card page-break-inside-card-featured">Page Break Inside Content</div>
```

## Classes
- `.page-break-inside-card`: Primary card with breathe animation
- `.page-break-inside-card-alt`: Secondary variant
- `.page-break-inside-card-ghost`: Ghost/outline card
- `.page-break-inside-card-glow`: Glow effect card
- `.page-break-inside-card-featured`: Gradient featured card
- `.page-break-inside-card-sm / -md / -lg`: Size variants
- `.page-break-inside-v-1 through -5`: Opacity variants
- `.page-break-inside-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.page-break-inside-hover-lift / -scale / -glow`: Hover effects
