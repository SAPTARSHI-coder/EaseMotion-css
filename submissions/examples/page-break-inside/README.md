# Page Break Inside

## Overview
Grid cell utilities for page-break-inside in EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/page-break-inside/style.css">
<div class="page-break-inside-wrap page-break-inside-cols-3 page-break-inside-gap-4">
  <div class="page-break-inside-cell">Page Break Inside Cell</div>
</div>
```

## Classes
- `.page-break-inside-wrap`: Grid container
- `.page-break-inside-cell / -alt / -acc / -out`: Cell variants
- `.page-break-inside-feat`: Gradient featured cell with shadow
- `.page-break-inside-cols-1 through -7`: Column count variants
- `.page-break-inside-gap-1 through -5`: Gap size variants
- `.page-break-inside-cell-hov / -feat-hov`: Hover animations
