# ✈️ Animated Airline Booking Dashboard

A modern, responsive **Airline Booking Dashboard** built using **HTML** and **CSS** with smooth animations.

## Features

- 📊 Overview Cards
  - Flights Today
  - Bookings
  - Passengers
  - Revenue

- 🛫 Flight Status Table

- 📈 Animated Boarding Progress Bars

- 🚪 Gate Information Grid

- 🔔 Flight Activity Feed

- ⚡ Quick Action Buttons
  - Book Flight
  - Check In Passenger
  - Download Report

- 📱 Responsive Design

- ✨ Smooth Hover & Fade Animations

## Technologies Used

- HTML5
- CSS3
- Font Awesome

## Folder Structure

```
animated-airline-dashboard/
│── demo.html
│── style.css
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `demo.html` in any modern web browser.
3. Explore the responsive airline dashboard.
