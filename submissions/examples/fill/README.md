# Fill Utilities

CSS utility classes for the `fill` property.

## Usage

```html
<div class="fill-none">...</div>
```

## Classes

- `.fill-none` ‚Äî fill: none;
- `.fill-current` ‚Äî fill: currentColor;
- `.fill-primary` ‚Äî fill: var(--ease-primary);
- `.fill-secondary` ‚Äî fill: var(--ease-secondary);
- `.fill-accent` ‚Äî fill: var(--ease-accent);
- `.fill-black` ‚Äî fill: #000;
- `.fill-white` ‚Äî fill: #fff;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
