# Font Variant Numeric

## Overview
List item and badge utilities for font-variant-numeric in EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/font-variant-numeric/style.css">
<li class="font-variant-numeric-item font-variant-numeric-item-on">
  Item content
  <span class="font-variant-numeric-badge font-variant-numeric-badge-pri">Label</span>
</li>
```

## Classes
- `.font-variant-numeric-item` through `.font-variant-numeric-item-13`: List items
- `.font-variant-numeric-badge-pri / -suc / -war / -out`: Badge variants
- `.font-variant-numeric-hl`: Text highlight effect
- `.font-variant-numeric-tag`: Uppercase tag label
- `.font-variant-numeric-pri-1 through -6`: Left border priority indicators
