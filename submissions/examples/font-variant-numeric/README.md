# Font Variant Numeric

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/font-variant-numeric/style.css">
<li class="font-variant-numeric-row font-variant-numeric-row-active">
  Font Variant Numeric item <span class="font-variant-numeric-tag">Font Variant Numeric</span>
</li>
```

## Classes
- `.font-variant-numeric-row`: List row with hover
- `.font-variant-numeric-row-active`: Active row with slide-in animation
- `.font-variant-numeric-tag / -alt / -outline / -success / -warn`: Tag variants
- `.font-variant-numeric-chip / -primary / -success`: Chip variants
- `.font-variant-numeric-hl`: Highlight text
- `.font-variant-numeric-prio-high / -med / -low`: Priority indicators
- `.font-variant-numeric-idx-1 through -3`: Index numbering styles
