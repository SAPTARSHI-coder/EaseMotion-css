Real CSS utilities using EaseMotion-css design tokens.

## Tokens Used
- ease-color-primary, ease-color-secondary, ease-color-success, ease-color-danger, ease-color-warning, ease-color-info
- ease-radius-sm, ease-radius-lg
- ease-shadow-sm, ease-shadow-md
- ease-space-2, ease-space-3, ease-space-4, ease-space-8
- ease-text-xs, ease-text-sm, ease-text-base, ease-text-lg, ease-text-xl
- ease-font-sans, ease-color-bg, ease-color-surface, ease-color-muted
- Dark mode and prefers-reduced-motion support
