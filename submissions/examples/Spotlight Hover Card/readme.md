# Animation: Spotlight Hover Card

A spotlight hover animation for card components. As the user moves their cursor across the card, a soft radial light follows the pointer, creating an interactive depth and focus effect.

## Features

* **Interactive spotlight effect**
* **Smooth cursor tracking**
* **Pure CSS rendering**
* **Minimal JavaScript**
* **Responsive layout**
* **Premium glassmorphism appearance**
* **Reduced motion support**

## Usage

```html
<div class="spotlight-card">
  <h3>Interactive Card</h3>
  <p>Move your cursor over this card.</p>
</div>
```

The spotlight position is controlled using CSS custom properties updated by JavaScript during mouse movement.
