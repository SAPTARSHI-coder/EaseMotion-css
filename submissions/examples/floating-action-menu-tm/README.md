# Floating Action Menu

## What does this do?
Adds `.ease-fab` — a floating action button that expands a menu on click.

```html
<div class="ease-fab">
  <button class="ease-fab-trigger">+</button>
  <div class="ease-fab-menu">
    <button>Action 1</button>
  </div>
</div>
```

## Preview
Open `demo.html` in your browser.
