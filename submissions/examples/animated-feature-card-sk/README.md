# Animated Feature Card

CSS-only feature card grid with hover lift and icon accents.

## Features

- 3-column responsive grid layout (adapts to 1-column on mobile)
- Color-coded icon circles (purple, blue, emerald, rose, amber, cyan)
- Hover lift effect with elevated shadow
- Icon scale-and-rotate animation on card hover
- Gradient accent bar that reveals on hover
- Gradient title with text clipping
- Animated fade-in entrance
- Custom properties for easy theming

## Usage

Include the `style.css` and use the BEM class structure. Each card needs `featurecard__icon--{color}` and `featurecard__accent--{color}` for proper theming.

## Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). Uses CSS grid, gradients, transforms, and custom properties.
