# XP Bar Fill Animation

## What does this do?
An experience bar that fills with an animated glow and shimmer effect.

## How is it used?
Add `.xp-bar` container with `.bar-track`, `.bar-fill`, and `.shine` elements.

## Why is it useful?
XP bars provide progression feedback in game UIs and gamified applications.
