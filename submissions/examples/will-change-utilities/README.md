# Will Change Utilities

## Overview
Grid cell utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/will-change-utilities/style.css">
<div class="will-change-utilities-grid-wrap will-change-utilities-cols-3 will-change-utilities-gap-2">
  <div class="will-change-utilities-cell">Will Change Utilities Cell</div>
</div>
```

## Classes
- `.will-change-utilities-cell`: Primary grid cell
- `.will-change-utilities-cell-alt`: Secondary variant
- `.will-change-utilities-cell-accent`: Accent variant
- `.will-change-utilities-cell-featured`: Gradient featured cell
- `.will-change-utilities-cell-outline`: Outline variant
- `.will-change-utilities-cell-sm / -md / -lg / -xl`: Size variants
- `.will-change-utilities-cols-2 / -3 / -4`: Column count
- `.will-change-utilities-gap-1 / -2 / -3`: Gap variants
- `.will-change-utilities-rwd-1 through -3`: Responsive padding
