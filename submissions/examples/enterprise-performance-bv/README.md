# Enterprise Performance Intelligence Platform

An enterprise-grade business intelligence and performance monitoring platform featuring SaaS metrics dashboards, corporate financial tables, strategy execution controls, and live sync log terminals. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Performance Header**: Suite branding logos, global status flags, and health levels.
- **SaaS Metric Cards**: Quad layout cards mapping Net Revenue Retention, Gross Margins, CAC Paybacks, and LTV:CAC ratios.
- **Financial Ledger Table**: Tables showing ARR forecasts, operational spendings, targets, and variance metrics.
- **BI Action Panel**: Controls triggering strategies adjustments and report compilers.
- **Sync Console Shell**: Log console compiling calculations.

## 🎨 Theme Details

- **Palette**: Dark Navy (`#0b0f19`), Surface Grey (`#161e2e`), Success Emerald (`#10b981`), Cyan Accent (`#0ea5e9`), Highlight Violet (`#7c3aed`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft sweeps, metric glow highlights, and transition buttons.
