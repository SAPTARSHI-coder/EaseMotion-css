# Text Justify Utilities

CSS utility classes for the `text-justify` property.

## Usage

```html
<div class="text-justify-auto">...</div>
```

- `.text-justify-intercharacter` тАФ text-justify: inter-character;
- `.text-justify-none` тАФ text-justify: none;
- `.text-justify-distribute` тАФ text-justify: distribute;
- `.text-justify-initial` тАФ text-justify: initial;
- `.text-justify-inherit` тАФ text-justify: inherit;
- `.text-justify-unset` тАФ text-justify: unset;
- `.justify-interword` тАФ text-justify: inter-word;
- `.justify-intercharacter` тАФ text-justify: inter-character;
- `.justify-distribute` тАФ text-justify: distribute;
- `.text-justify-normal` тАФ text-justify: normal;
- `.text-justify-revert` тАФ text-justify: revert;
- `.justify-words` тАФ text-justify: inter-word;
