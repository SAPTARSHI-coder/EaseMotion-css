# Multi-Step Onboarding Wizard Component

A modern multi-step onboarding and registration workflow component built using pure HTML and CSS.

## Features

- Multi-step onboarding flow UI
- Animated progress indicator
- Completed, active, and pending states
- Responsive design
- Form sections for onboarding
- Smooth transitions and hover effects
- Enterprise-style workflow layout
- Accessibility-friendly structure

- demo.html
- style.css

- User Registration Flows
- SaaS Onboarding Systems
- Surveys & Questionnaires
- Enterprise Applications
- Setup Wizards
- Product Walkthroughs
