Relative Color Syntax

Manipulating color channels using the from keyword

#60a5fa
#...

Lighter

Lightness increased by 0.15

oklch(from var(--base) calc(l + 0.15) c h)
#...

Darker

Lightness decreased by 0.15

oklch(from var(--base) calc(l - 0.15) c h)
#...

Complementary

Hue shifted by +180°

oklch(from var(--base) l c calc(h + 180))
#...

Transparent

Alpha channel set to 40%

oklch(from var(--base) l c h / 0.4)
