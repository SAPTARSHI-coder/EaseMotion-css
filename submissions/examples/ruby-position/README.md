# Ruby Position

## Overview
List row and tag utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/ruby-position/style.css">
<li class="ruby-position-row ruby-position-row-active">
  Ruby Position item <span class="ruby-position-tag">Ruby Position</span>
</li>
```

## Classes
- `.ruby-position-row`: List row with hover
- `.ruby-position-row-active`: Active row with slide-in animation
- `.ruby-position-tag / -alt / -outline / -success / -warn`: Tag variants
- `.ruby-position-chip / -primary / -success`: Chip variants
- `.ruby-position-hl`: Highlight text
- `.ruby-position-prio-high / -med / -low`: Priority indicators
- `.ruby-position-idx-1 through -3`: Index numbering styles
