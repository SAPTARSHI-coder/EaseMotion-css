# Animated FAQ Accordion Component

## Description

A lightweight CSS-only FAQ accordion with smooth expand animations, rotating icons, and elegant hover interactions.

---

## Features

- Pure CSS implementation using `<details>` and `<summary>`
- Smooth expand and collapse animations
- Rotating plus icon animation
- Hover lift effect with shadows
- Responsive design for mobile devices
- No JavaScript required
- Reduced motion accessibility support

---

## Classes

- `.faq-section` — Main FAQ container
- `.faq-container` — Wrapper for FAQ items
- `.faq-item` — Individual accordion item
- `.icon` — Rotating expand icon
- `.answer` — Accordion content

---

## Accessibility

Uses semantic HTML elements and supports `prefers-reduced-motion` for users who prefer fewer animations.
