Shape Outside Wrapping

Magazine-style text wrapping using shape-outside with clip-path — no JavaScript required. Each article floats a coloured shape and wraps text around its contour.

Circle circle(50%)

CSS shapes let text wrap around circular elements naturally, without JavaScript or complex markup. The shape-outside: circle(50%) property defines the wrapping contour as a perfect circle, while clip-path: circle(50%) ensures the visible element matches the same boundary. A shape-margin of 1rem adds comfortable breathing room between the shape edge and the flowing text.

shape-outside: circle(50%)

Ellipse ellipse(80px 120px)

Elliptical shapes create a wider, asymmetric wrapping path ideal for portrait images or decorative ovals. The shape-outside: ellipse(80px 120px at 50% 50%) declaration defines a horizontal radius of 80px and vertical radius of 120px, producing a tall oval. Text flows smoothly around the curved boundary, creating a polished editorial layout.

shape-outside: ellipse(80px 120px at 50% 50%)

Polygon polygon(…)

Custom polygons enable wrapping around arbitrary shapes — diagonal cuts, slants, or angled edges. This example uses polygon(0 0, 100% 0, 100% 100%, 0 75%) to create a diagonal bottom edge. The text follows the slant, producing a dynamic layout that breaks away from rectangular convention. Polygons are defined as a set of x/y coordinate pairs, giving pixel-perfect control over the wrapping contour.

shape-outside: polygon(0 0, 100% 0, 100% 100%, 0 75%)

Polygon (star) polygon(…)

A star-like polygon demonstrates complex wrapping with multiple concave and convex points. The 10-point star shape is defined by alternating radii, wrapping text on both the left and right sides. The float: right placement shows that shapes can wrap from either direction, giving designers flexibility to create balanced, asymmetrical magazine spreads.

shape-outside: polygon(50% 0, 80% 30%, …)

Inset inset(0 round 30%)

The inset() function creates a rounded rectangle wrapping path. Here inset(0 round 30%) keeps the shape at full size while rounding each corner by 30% of the element's dimensions. This is ideal for avatar images, product cards, or any UI element where text should flow around a softly rounded container rather than a hard-edged box.

shape-outside: inset(0 round 30%)
All shapes use shape-margin: 1rem for spacing. The clip-path property mirrors the shape-outside value so the visible gradient element matches the wrapping contour exactly.
