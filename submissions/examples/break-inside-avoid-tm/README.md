`break-inside-avoid-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/break-inside-avoid-tm/style.css">
<div class="break-inside-avoid-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .break-inside-avoid: prevents break within element

## Impact

Prevents multi-column layout from breaking cards
