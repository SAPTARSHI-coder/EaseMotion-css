# Container Query Wrapper

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/container-query-wrapper/style.css">
<div class="container-query-wrapper-card container-query-wrapper-card-featured">Container Query Wrapper Content</div>
```

## Classes
- `.container-query-wrapper-card`: Primary card with breathe animation
- `.container-query-wrapper-card-alt`: Secondary variant
- `.container-query-wrapper-card-ghost`: Ghost/outline card
- `.container-query-wrapper-card-glow`: Glow effect card
- `.container-query-wrapper-card-featured`: Gradient featured card
- `.container-query-wrapper-card-sm / -md / -lg`: Size variants
- `.container-query-wrapper-v-1 through -5`: Opacity variants
- `.container-query-wrapper-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.container-query-wrapper-hover-lift / -scale / -glow`: Hover effects

## Design
