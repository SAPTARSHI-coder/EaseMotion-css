# Box Decoration Break Utilities

CSS utility classes for controlling how decorations render across element fragments.

## Classes

- `.box-decoration-clone` — `box-decoration-break: clone`
- `.box-decoration-slice` — `box-decoration-break: slice`

## Why

When an inline element wraps across multiple lines, `clone` applies borders, padding, and backgrounds to each fragment individually. Useful for highlighted inline text spans that wrap.
