# Loading Spinner (cv variant)

## What does this do?
A collection of pure CSS animated loading spinners with 4 variants (Classic, Dots, Pulse, Gradient), 3 sizes, and 4 color options. Perfect for loading states, async operations, and processing indicators.

## How is it used?

### Basic Usage:

```html
<!-- Classic spinner -->
<div class="ease-spinner-cv ease-spinner-cv-classic"></div>

<!-- Dots spinner -->
<div class="ease-spinner-cv ease-spinner-cv-dots">
    <span></span><span></span><span></span>
</div>

<!-- Pulse spinner -->
<div class="ease-spinner-cv ease-spinner-cv-pulse"></div>

<!-- Gradient spinner -->
<div class="ease-spinner-cv ease-spinner-cv-gradient"></div>

<!-- Small (24px) -->
<div class="ease-spinner-cv ease-spinner-cv-classic ease-spinner-cv-sm"></div>

<!-- Medium (40px - default) -->
<div class="ease-spinner-cv ease-spinner-cv-classic"></div>

<!-- Large (60px) -->
<div class="ease-spinner-cv ease-spinner-cv-classic ease-spinner-cv-lg"></div>

<div class="ease-spinner-cv ease-spinner-cv-classic ease-spinner-cv-primary"></div>
<div class="ease-spinner-cv ease-spinner-cv-classic ease-spinner-cv-info"></div>
<div class="ease-spinner-cv ease-spinner-cv-classic ease-spinner-cv-success"></div>
<div class="ease-spinner-cv ease-spinner-cv-classic ease-spinner-cv-white"></div>
