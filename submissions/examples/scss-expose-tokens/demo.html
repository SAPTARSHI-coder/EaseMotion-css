2. Consuming Tokens in JavaScript & React

Here is how developers can import these SCSS tokens directly inside React (e.g. Framer Motion, GSAP, or canvas animation timings matching CSS speeds):

Export definition (scss/_exports.scss) 
@use 'variables';

:export {
  speedFast:        variables.$speed-fast;
  speedMedium:      variables.$speed-medium;
  easeBounce:       variables.$ease-bounce;
}
React Component consumption (MotionCard.jsx) 
import React from 'react';
import { motion } from 'framer-motion';
// Import exportable CSS Module tokens
import tokens from 'easemotion-css/scss/index.module.scss';

export function MotionCard() {
  return (
    <motion.div
      animate={{ scale: 1.1 }}
      transition={{ 
        // Framer Motion timing values aligned with CSS speed tokens
        duration: parseFloat(tokens.speedMedium) / 1000, 
        ease: [0.34, 1.56, 0.64, 1] // parsed from tokens.easeBounce
      }}
    >
      Hover Me
    </motion.div>
  );
}
