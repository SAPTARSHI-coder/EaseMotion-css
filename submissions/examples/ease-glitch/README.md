# Ease Glitch Utility

A pure CSS glitch effect that creates a retro digital distortion animation on text or elements.

## Features
- **Zero JavaScript:** Relies entirely on CSS keyframes, `clip-path`, and `transform`.
- **Customizable Speed:** Use `.glitch-slow` or `.glitch-fast` modifiers.
- **Easy Integration:** Single class implementation.

## How to use
Add the `glitch` class to your element:
```html
<h1 class="glitch">Your Text Here</h1>
