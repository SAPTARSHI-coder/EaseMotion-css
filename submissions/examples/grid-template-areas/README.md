# Grid Template Areas

## Overview
Grid cell utilities for grid-template-areas in EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/grid-template-areas/style.css">
<div class="grid-template-areas-wrap grid-template-areas-cols-3 grid-template-areas-gap-4">
  <div class="grid-template-areas-cell">Grid Template Areas Cell</div>
</div>
```

## Classes
- `.grid-template-areas-wrap`: Grid container
- `.grid-template-areas-cell / -alt / -acc / -out`: Cell variants
- `.grid-template-areas-feat`: Gradient featured cell with shadow
- `.grid-template-areas-cols-1 through -7`: Column count variants
- `.grid-template-areas-gap-1 through -5`: Gap size variants
- `.grid-template-areas-cell-hov / -feat-hov`: Hover animations
