# Grid Template Areas

## Overview
Grid cell utilities for EaseMotion CSS.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/grid-template-areas/style.css">
<div class="grid-template-areas-grid-wrap grid-template-areas-cols-3 grid-template-areas-gap-2">
  <div class="grid-template-areas-cell">Grid Template Areas Cell</div>
</div>
```

## Classes
- `.grid-template-areas-cell`: Primary grid cell
- `.grid-template-areas-cell-alt`: Secondary variant
- `.grid-template-areas-cell-accent`: Accent variant
- `.grid-template-areas-cell-featured`: Gradient featured cell
- `.grid-template-areas-cell-outline`: Outline variant
- `.grid-template-areas-cell-sm / -md / -lg / -xl`: Size variants
- `.grid-template-areas-cols-2 / -3 / -4`: Column count
- `.grid-template-areas-gap-1 / -2 / -3`: Gap variants
- `.grid-template-areas-rwd-1 through -3`: Responsive padding
