Paint Order

paint-order — controlling text fill and stroke rendering order

The paint-order CSS property controls whether text fill or stroke is drawn first. With paint-order: stroke fill, the stroke renders behind the fill, creating clean outlined text. The default normal (fill then stroke) often makes thick strokes look jagged.
