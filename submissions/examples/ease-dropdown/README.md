# ease-dropdown

A CSS-only dropdown menu with hover and focus support, featuring a smooth fade-and-slide opening animation.

## Features

- `.ease-dropdown` wrapper
- `.ease-dropdown-toggle` trigger
- `.ease-dropdown-menu` content container
- Hover support
- Keyboard focus support
- Fade-and-slide animation
- Pure CSS implementation
- No JavaScript required

## Usage

```html
<div class="ease-dropdown">
    <button class="ease-dropdown-toggle">
        Menu
    </button>

    <ul class="ease-dropdown-menu">
        <li><a href="#">Dashboard</a></li>
        <li><a href="#">Projects</a></li>
        <li><a href="#">Settings</a></li>
    </ul>
</div>
```

## Why it fits EaseMotion CSS

The component adds meaningful motion through a subtle fade-and-slide animation while keeping the implementation lightweight, readable, and framework-independent.
