# interpolate-size: allow-keywords

Demonstrates `interpolate-size: allow-keywords` for smooth CSS transitions between `height: auto` and explicit values, enabling natural accordion and expand-section animations without `calc-size()`.

## Features

- Four-item accordion with smooth height: 0 ↔ height: auto transitions
- Expandable content panel with dynamic inner content
- Side-by-side comparison with and without `interpolate-size: allow-keywords`
- Works without JavaScript height measuring or calc-size() wrappers

## Usage

Open `demo.html` in a browser supporting `interpolate-size: allow-keywords` (Chrome 129+, Edge 129+). Click accordion headers and toggle buttons to see smooth height transitions.
