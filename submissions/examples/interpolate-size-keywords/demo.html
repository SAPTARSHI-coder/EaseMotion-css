✧ interpolate-size: allow-keywords

Smooth height: auto transitions without calc-size()

interpolate-size: allow-keywords lets you transition between auto and explicit lengths. Open the accordions below to see smooth height expansion without JavaScript measuring or calc-size().

Accordion

Each section transitions height from 0 to auto

interpolate-size: allow-keywords is a CSS property that enables transitions to and from keyword length values like auto, min-content, max-content, and fit-content.

Previously, CSS could not animate between auto and a fixed value because auto is a keyword, not a numeric length.

calc-size() is a function that allows you to perform calculations with intrinsic sizes. interpolate-size is a simpler declaration that globally enables keyword interpolation.

With interpolate-size: allow-keywords, you can write straightforward transitions without wrapping values in calc-size().

As of 2025, interpolate-size: allow-keywords is supported in Chrome 129+, Edge 129+, and Safari 18.2+. Firefox support is in development.

Always check caniuse.com for the latest browser compatibility data.

CSS Setup

/* Enable on the element or globally */
.panel {
  interpolate-size: allow-keywords;
  transition: height 0.4s ease;
  overflow: hidden;
}

.panel.collapsed {
  height: 0;
}

.panel:not(.collapsed) {
  height: auto;
}
