How it works

  • .field:has(input:focus) — highlights the entire field group when an input is focused.
  • .field:has(input:invalid:not(:placeholder-shown)) — shows error state when input is invalid and touched.
  • .field:has(input:valid:not(:placeholder-shown)) — shows success state.
  • .field-group--checkbox:has(input:checked) — removes error styling when any checkbox is checked.
  • All styling is purely CSS — no JavaScript validation classes needed.
