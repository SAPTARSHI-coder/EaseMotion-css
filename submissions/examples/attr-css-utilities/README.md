# Attr Css Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/attr-css-utilities/style.css">
<div class="attr-css-utilities-card attr-css-utilities-card-featured">Attr Css Utilities Content</div>
```

## Classes
- `.attr-css-utilities-card`: Primary card with breathe animation
- `.attr-css-utilities-card-alt`: Secondary variant
- `.attr-css-utilities-card-ghost`: Ghost/outline card
- `.attr-css-utilities-card-glow`: Glow effect card
- `.attr-css-utilities-card-featured`: Gradient featured card
- `.attr-css-utilities-card-sm / -md / -lg`: Size variants
- `.attr-css-utilities-v-1 through -5`: Opacity variants
- `.attr-css-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.attr-css-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 216deg
- Border radius: 7px / 13px / 24px
