# Relative Color

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/relative-color/style.css">
<div class="relative-color-card relative-color-card-featured">Relative Color Content</div>
```

## Classes
- `.relative-color-card`: Primary card with breathe animation
- `.relative-color-card-alt`: Secondary variant
- `.relative-color-card-ghost`: Ghost/outline card
- `.relative-color-card-glow`: Glow effect card
- `.relative-color-card-featured`: Gradient featured card
- `.relative-color-card-sm / -md / -lg`: Size variants
- `.relative-color-v-1 through -5`: Opacity variants
- `.relative-color-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.relative-color-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 295deg
- Border radius: 7px / 13px / 18px
