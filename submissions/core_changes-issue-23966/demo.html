ease-image

CSS image-hint utilities that pair with HTML loading, decoding, and fetchpriority attributes.

Note: These CSS classes provide browser hints via content-visibility. Always pair them with the corresponding HTML attributes for full effect.

Lazy Load — .ease-image-lazy

Off-screen images: defers load + decode.

▶
lazy

Deferred Image

loading="lazy" decoding="async"

Eager — .ease-image-eager

Above-fold images: immediately visible.

▶
eager

Instant Image

loading="eager" decoding="sync"

Priority — .ease-image-priority

Hero / LCP images: highest fetch priority.

★
priority

Hero Image

fetchpriority="high"

Async Decode — .ease-image-async

Decode off the critical path.

↻
async

Async Decode

decoding="async"

Sync Decode — .ease-image-sync

For critical above-fold images.

▶
sync

Sync Decode

decoding="sync"

Recommended Usage

<!-- Hero / LCP — highest priority, sync decode -->
<img class="ease-image-priority"
src="hero.jpg" fetchpriority="high"
decoding="sync" alt="">

<!-- Off-screen — deferred load + decode -->
<img class="ease-image-lazy"
src="photo.jpg" loading="lazy"
decoding="async" alt="">

<!-- Above-fold — eager load -->
<img class="ease-image-eager"
src="banner.png" loading="eager"
alt="">
